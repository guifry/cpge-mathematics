Continuous Linear Maps and the Operator Norm

Phase 14 — Normed Vector Spaces, Page 4

The problem

In finite dimensions, every linear map $T : \mathbb{R}^n \to \mathbb{R}^m$ is continuous. This is a theorem you probably take for granted: matrix multiplication cannot "blow up" a convergent sequence. But in infinite dimensions, this fails. There exist linear maps that send convergent sequences to divergent ones — the differentiation operator $D : C^1([0,1]) \to C([0,1])$, for instance, is unbounded under the supremum norm.

The question becomes: which linear maps between normed spaces are continuous? The answer, due to Riesz and Banach, is clean: exactly the bounded ones. And the "size" of such a map is measured by a single number — the operator norm.

1. Continuity equals boundedness

Theorem — Continuity criterion for linear maps

Let $T : X \to Y$ be a linear map between normed spaces. The following are equivalent:

(i) $T$ is continuous.

(ii) $T$ is continuous at $0$.

(iii) $T$ is bounded: there exists $C \geq 0$ with $\|Tx\| \leq C\|x\|$ for all $x \in X$.

Proof

Step 1. (i) $\Rightarrow$ (ii). Trivial: continuous everywhere implies continuous at $0$.

Step 2. (ii) $\Rightarrow$ (iii). Continuous at $0$ means: for $\varepsilon = 1$ there exists $\delta > 0$ with $\|Tx\| < 1$ whenever $\|x\| < \delta$. For arbitrary $x \neq 0$, set $u = \delta x / (2\|x\|)$. Then $\|u\| = \delta/2 < \delta$, so $\|Tu\| < 1$. By linearity, $\|Tx\| = (2\|x\|/\delta)\|Tu\| < 2\|x\|/\delta$. Taking $C = 2/\delta$ gives $\|Tx\| \leq C\|x\|$ (this holds for $x = 0$ trivially).

Step 3. (iii) $\Rightarrow$ (i). For any $x, y \in X$: $\|Tx - Ty\| = \|T(x - y)\| \leq C\|x - y\|$. So $T$ is Lipschitz continuous, hence continuous everywhere.

∎

The key insight is Step 3: linearity upgrades continuity at a single point to Lipschitz continuity everywhere. This is special to linear maps — for general functions, continuity at one point says nothing about other points.

2. The operator norm

Definition — Operator norm

For a bounded linear map $T : X \to Y$, define

$$\|T\| = \sup_{\|x\| \leq 1} \|Tx\| = \sup_{\|x\| = 1} \|Tx\| = \sup_{x \neq 0} \frac{\|Tx\|}{\|x\|}.$$
Proof that the three expressions are equal

Step 1. Call the three quantities $A$, $B$, $C$ respectively. Since $\{x : \|x\| = 1\} \subset \{x : \|x\| \leq 1\}$, we have $B \leq A$.

Step 2. For any $x$ with $\|x\| \leq 1$ and $x \neq 0$, set $u = x/\|x\|$. Then $\|u\| = 1$ and $\|Tx\| = \|x\|\,\|Tu\| \leq \|Tu\| \leq B$. Also $\|T(0)\| = 0 \leq B$. So $A \leq B$, giving $A = B$.

Step 3. For $x \neq 0$, $\|Tx\|/\|x\| = \|T(x/\|x\|)\|$, and $x/\|x\|$ has norm 1. So $C = B$.

∎

The operator norm is the smallest constant $C$ that works in $\|Tx\| \leq C\|x\|$. It measures the maximum "stretching factor" of $T$.

Proof that $\|\cdot\|$ is a norm on bounded linear maps

Step 1. Positive definiteness: $\|T\| \geq 0$ always. If $\|T\| = 0$, then $\|Tx\| \leq 0$ for all $x$ with $\|x\| = 1$, so $Tx = 0$ for all unit vectors, hence $T = 0$ by linearity.

Step 2. Homogeneity: $\|\lambda T\| = \sup_{\|x\|=1} \|\lambda Tx\| = |\lambda| \sup_{\|x\|=1} \|Tx\| = |\lambda|\,\|T\|$.

Step 3. Triangle inequality: $\|T + S\| = \sup_{\|x\|=1} \|(T+S)x\| \leq \sup_{\|x\|=1} (\|Tx\| + \|Sx\|) \leq \|T\| + \|S\|$.

∎

3. The space $\mathcal{L}(X, Y)$ is Banach when $Y$ is

Write $\mathcal{L}(X, Y)$ for the vector space of all bounded linear maps from $X$ to $Y$, equipped with the operator norm.

Theorem

If $Y$ is a Banach space, then $\mathcal{L}(X, Y)$ is a Banach space.

Proof

Step 1. Let $(T_n)$ be Cauchy in $\mathcal{L}(X, Y)$. For each $x \in X$, $\|T_m x - T_n x\| \leq \|T_m - T_n\|\,\|x\| \to 0$, so $(T_n x)$ is Cauchy in $Y$.

Step 2. Since $Y$ is complete, define $Tx = \lim_{n \to \infty} T_n x$ for each $x$.

Step 3. $T$ is linear: $T(\alpha x + \beta y) = \lim T_n(\alpha x + \beta y) = \alpha \lim T_n x + \beta \lim T_n y = \alpha Tx + \beta Ty$.

Step 4. $T$ is bounded: since $(T_n)$ is Cauchy, $\|T_n\|$ is bounded (say by $M$). For any $x$, $\|Tx\| = \lim \|T_n x\| \leq M\|x\|$, so $\|T\| \leq M$.

Step 5. $T_n \to T$ in operator norm: given $\varepsilon > 0$, choose $N$ so that $\|T_m - T_n\| < \varepsilon$ for $m, n \geq N$. For any $x$ with $\|x\| \leq 1$, $\|T_m x - T_n x\| \leq \varepsilon$. Let $m \to \infty$: $\|Tx - T_n x\| \leq \varepsilon$. Taking the supremum over $\|x\| \leq 1$: $\|T - T_n\| \leq \varepsilon$ for all $n \geq N$.

∎

4. The dual space

A particularly important case: linear maps from $X$ to the scalars $\mathbb{R}$.

Definition — Dual space

The dual space of $X$ is $X^* = \mathcal{L}(X, \mathbb{R})$, the space of all bounded linear functionals $\varphi : X \to \mathbb{R}$.

Since $\mathbb{R}$ is complete, the theorem of Section 3 gives us immediately: $X^*$ is always a Banach space, even if $X$ itself is not complete. This is a remarkable fact — the dual of an incomplete normed space is complete.

Finite-dimensional example. If $X = \mathbb{R}^n$ with the Euclidean norm, then every linear functional has the form $\varphi(x) = a \cdot x$ for some $a \in \mathbb{R}^n$, and $\|\varphi\| = \|a\|_2$. So $(\mathbb{R}^n)^* \cong \mathbb{R}^n$ (this is the Riesz representation theorem for inner product spaces — Phase 13, page 1).

Sequence spaces. For $1 < p < \infty$ with $1/p + 1/q = 1$, we have $(\ell^p)^* \cong \ell^q$: every bounded linear functional on $\ell^p$ is given by $\varphi((x_n)) = \sum a_n x_n$ for a unique $(a_n) \in \ell^q$, with $\|\varphi\| = \|(a_n)\|_q$. We state this without proof; it requires Hölder's inequality and a careful density argument.

5. In finite dimensions, all linear maps are continuous

Theorem

If $X$ is a finite-dimensional normed space and $Y$ is any normed space, then every linear map $T : X \to Y$ is bounded (hence continuous).

Proof

Step 1. Let $\dim X = n$ with basis $(e_1, \ldots, e_n)$. For $x = \sum_{i=1}^{n} \alpha_i e_i$:

$$\|Tx\| = \left\|\sum_{i=1}^{n} \alpha_i T e_i\right\| \leq \sum_{i=1}^{n} |\alpha_i|\,\|T e_i\|.$$

Step 2. Set $M = \max_i \|T e_i\|$. Then $\|Tx\| \leq M \sum |\alpha_i| = M \|\alpha\|_1$ where $\alpha = (\alpha_1, \ldots, \alpha_n)$.

Step 3. By norm equivalence (Phase 14, page 1), there exists $K > 0$ with $\|\alpha\|_1 \leq K \|x\|$ (since $x \mapsto \alpha$ is the coordinate map, and $\|\cdot\|_1$ on coordinates is equivalent to $\|\cdot\|$ on $X$). So $\|Tx\| \leq MK\|x\|$, giving $\|T\| \leq MK$.

∎

This is why linear algebra in $\mathbb{R}^n$ feels so smooth: continuity is automatic. In infinite dimensions, you must earn continuity by proving boundedness.

Historical note

The equivalence of continuity and boundedness for linear maps was established by Riesz (1918) and Banach (1920s). The operator norm was implicit in Hilbert's work on integral equations (1904–1910), where he studied bounded bilinear forms on $\ell^2$. The concept of the dual space traces to Hans Hahn (1927), who proved the first version of the Hahn–Banach extension theorem — the tool that guarantees $X^*$ is "large enough" to separate points.

What breaks

Without linearity. A bounded nonlinear function need not be continuous (consider the characteristic function of $\mathbb{Q}$: bounded but discontinuous everywhere). Conversely, a continuous nonlinear function need not be bounded ($f(x) = x$ on $\mathbb{R}$). The equivalence of boundedness and continuity is a privilege of linearity.

Without completeness of $Y$. The space $\mathcal{L}(X, Y)$ need not be complete. You lose the ability to take limits of operator sequences — a Cauchy sequence of bounded operators might "converge" to something outside the space. This is why Banach space targets are standard in operator theory.

Without finite dimension. The differentiation operator $D : (C^1([0,1]), \|\cdot\|_\infty) \to (C([0,1]), \|\cdot\|_\infty)$ is linear but unbounded: $\|D(x^n)\|_\infty = n$ while $\|x^n\|_\infty = 1$. There is no constant $C$ with $\|Df\|_\infty \leq C\|f\|_\infty$.

A linear map between normed spaces is continuous iff it is bounded, and the operator norm measures its worst-case stretching.
