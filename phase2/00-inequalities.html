Inequalities

Phase 2 — The Infinite: Limits and Convergence, Page 0

The problem

You want to prove that a sequence converges. The definition says: for every $\varepsilon > 0$, find $N$ such that $|a_n - L| < \varepsilon$ for all $n > N$. Every step of this argument is an inequality. Bounding $|a_n - L|$ requires chaining, splitting, and estimating expressions — and a single wrong direction kills the proof.

Inequality manipulation is the engine of analysis. Before we touch limits, we need the tools.

1. Basic rules

Rules of inequality

For $a, b, c \in \mathbb{R}$:

  • Addition preserves order: $a < b \;\Longrightarrow\; a + c < b + c$
  • Positive scaling preserves order: $a < b$ and $c > 0 \;\Longrightarrow\; ac < bc$
  • Negative scaling reverses order: $a < b$ and $c < 0 \;\Longrightarrow\; ac > bc$
  • Transitivity: $a < b$ and $b < c \;\Longrightarrow\; a < c$

The third rule is where most errors happen. Dividing both sides by a negative number flips the inequality.

2. Absolute value

Definition
$$|a| = \begin{cases} a & \text{if } a \geq 0 \\ -a & \text{if } a < 0 \end{cases}$$

Equivalently: $|a| = \sqrt{a^2}$. This means $|a| \geq 0$ always, and $|a| = 0 \Leftrightarrow a = 0$.

The key equivalence for analysis: $|a - b| < \varepsilon$ means $a$ is within $\varepsilon$ of $b$. This is the language of limits.

Triangle inequality

For all $a, b \in \mathbb{R}$:

$$|a + b| \leq |a| + |b|$$
Proof

We have $-|a| \leq a \leq |a|$ and $-|b| \leq b \leq |b|$. Adding: $-(|a|+|b|) \leq a+b \leq |a|+|b|$. This is exactly $|a+b| \leq |a|+|b|$.

∎
Reverse triangle inequality
$$\big||a| - |b|\big| \leq |a - b|$$
Proof

Write $a = (a - b) + b$. By the triangle inequality, $|a| \leq |a - b| + |b|$, so $|a| - |b| \leq |a - b|$. Swap $a$ and $b$: $|b| - |a| \leq |b - a| = |a - b|$. Combining: $\big||a| - |b|\big| \leq |a - b|$.

∎

The reverse triangle inequality is used constantly in limit proofs to bound $|f(x)| - |f(a)|$ by $|f(x) - f(a)|$.

3. AM-GM inequality

Arithmetic mean — Geometric mean

For $a, b \geq 0$:

$$\frac{a + b}{2} \geq \sqrt{ab}$$

Equality holds if and only if $a = b$.

Proof

Start from a fact that's always true: $(\sqrt{a} - \sqrt{b})^2 \geq 0$. Expand: $a - 2\sqrt{ab} + b \geq 0$. Rearrange: $a + b \geq 2\sqrt{ab}$. Divide by 2.

Equality iff $\sqrt{a} = \sqrt{b}$ iff $a = b$.

∎

AM-GM is the first tool for bounding products in terms of sums (and vice versa). It appears in optimisation problems, convergence proofs, and information theory.

4. Bernoulli's inequality

Bernoulli's inequality

For $x \geq -1$ and $n \in \mathbb{N}$:

$$(1 + x)^n \geq 1 + nx$$
Proof (by induction)

Base case: $n = 1$: $(1+x)^1 = 1 + x$. True.

Inductive step: assume $(1+x)^k \geq 1 + kx$. Then:

$$(1+x)^{k+1} = (1+x)^k(1+x) \geq (1+kx)(1+x)$$

The last step uses the inductive hypothesis and $1 + x \geq 0$ (so multiplying preserves the inequality). Expanding:

$$(1+kx)(1+x) = 1 + (k+1)x + kx^2 \geq 1 + (k+1)x$$

since $kx^2 \geq 0$.

∎

Bernoulli's inequality is the standard tool for bounding exponential growth from below. It's essential for proving that sequences like $(1 + 1/n)^n$ are bounded.

5. Cauchy-Schwarz inequality

Cauchy-Schwarz (finite version)

For real numbers $a_1, \ldots, a_n$ and $b_1, \ldots, b_n$:

$$\left(\sum_{i=1}^n a_i b_i\right)^2 \leq \left(\sum_{i=1}^n a_i^2\right)\left(\sum_{i=1}^n b_i^2\right)$$
Proof

For any $t \in \mathbb{R}$, consider $\sum_{i=1}^n (a_i + tb_i)^2 \geq 0$. Expanding:

$$\sum a_i^2 + 2t\sum a_i b_i + t^2 \sum b_i^2 \geq 0$$

This is a quadratic in $t$ that is always $\geq 0$. A quadratic $At^2 + Bt + C \geq 0$ for all $t$ iff its discriminant $B^2 - 4AC \leq 0$. Here $A = \sum b_i^2$, $B = 2\sum a_i b_i$, $C = \sum a_i^2$. So:

$$4\left(\sum a_i b_i\right)^2 - 4\left(\sum a_i^2\right)\left(\sum b_i^2\right) \leq 0$$

which is exactly the Cauchy-Schwarz inequality.

∎

This reappears in Phase 6 (inner products) and Phase 12 (covariance). The abstract version — $|\langle u, v \rangle| \leq \|u\|\|v\|$ — is the foundation of geometry in any inner product space.

6. Chaining inequalities

The core skill in epsilon-delta proofs is chaining: bound each piece separately, then combine.

Example: show $|a_n b_n - ab| \to 0$ given $a_n \to a$ and $b_n \to b$.

The trick: insert a middle term. $a_n b_n - ab = a_n b_n - a_n b + a_n b - ab = a_n(b_n - b) + b(a_n - a)$.

By the triangle inequality: $|a_n b_n - ab| \leq |a_n||b_n - b| + |b||a_n - a|$.

Since $a_n \to a$, the sequence $(a_n)$ is bounded: $|a_n| \leq M$ for some $M$. So $|a_n b_n - ab| \leq M|b_n - b| + |b||a_n - a|$. Both terms go to $0$.

This pattern — split, triangle inequality, bound each piece — is used on nearly every page of Phase 2.

Historical note

The triangle inequality gets its name from geometry: in a triangle, the sum of any two sides exceeds the third. Cauchy published the finite sum version of Cauchy-Schwarz in 1821 in his Cours d'analyse. Schwarz extended it to integrals in 1885. Bernoulli's inequality is attributed to Jacob Bernoulli (1689), though the proof by induction is standard fare from Euler onwards. AM-GM was known to the ancient Greeks — Euclid proves a geometric version in the Elements (Book II, Proposition 5).

What breaks

Triangle inequality with the wrong direction: $|a + b| \geq |a| + |b|$ is false. Counterexample: $a = 1$, $b = -1$. Then $|a+b| = 0 < 2 = |a| + |b|$.

Bernoulli without $x \geq -1$: try $x = -3$, $n = 4$. Then $(1+x)^4 = (-2)^4 = 16$ but $1 + 4(-3) = -11$. Here $16 \geq -11$ still holds accidentally. The real failure: $x = -1.5$, $n = 3$. $(1-1.5)^3 = (-0.5)^3 = -0.125$ but $1 + 3(-1.5) = -3.5$. We get $-0.125 \geq -3.5$, which holds. Try $n = 2$: $(1-1.5)^2 = 0.25$ but $1 + 2(-1.5) = -2$. Still holds. The inequality is hard to break numerically, but the proof breaks: the inductive step multiplies by $(1+x)$, which is negative when $x < -1$. Multiplying an inequality by a negative number reverses it, so the chain of reasoning collapses. Without $x \geq -1$ the proof is invalid, even if specific counterexamples are hard to exhibit for small $n$.

AM-GM with negative numbers: $a = -4$, $b = -1$. Then $(a+b)/2 = -5/2$ and $\sqrt{ab} = 2$. The inequality $-5/2 \geq 2$ is false. AM-GM requires $a, b \geq 0$.

Every convergence proof is a chain of inequalities — split, bound each piece, reassemble.
