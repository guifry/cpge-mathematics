Gronwall's Inequality

Phase 8 — Differential Equations, Page 3

The problem

You know a solution to an ODE exists and is unique (page 2). But how fast can it grow? If you perturb the initial condition slightly, does the solution change by a little or a lot?

These questions reduce to a single pattern: you have a non-negative function $u(t)$ that satisfies an integral inequality — $u$ is bounded by $\alpha$ plus its own integral — and you need a bound on $u$ itself. Gronwall (1919) proved that such a function grows at most exponentially. This one inequality is the master tool for stability, uniqueness, and continuous dependence in ODE theory.

1. The setup

Suppose $u(t) \geq 0$ satisfies an inequality of the form

$$u(t) \leq \alpha + \int_{t_0}^t \beta(s)\, u(s)\, ds$$

for $t \in [t_0, T]$, where $\alpha \geq 0$ is a constant and $\beta(s) \geq 0$ is a continuous function. The right side involves $u$ itself inside the integral — so this is an implicit bound. We want an explicit bound: $u(t) \leq$ something involving only $\alpha$, $\beta$, and $t$.

The naïve approach — substitute the inequality back into itself — yields an infinite chain. Gronwall's inequality cuts through this by giving the exact answer in one step.

2. The inequality

Theorem — Gronwall's inequality (integral form)

Let $u, \beta : [t_0, T] \to [0, \infty)$ be continuous. Let $\alpha \geq 0$. If

$$u(t) \leq \alpha + \int_{t_0}^t \beta(s)\, u(s)\, ds \quad \text{for all } t \in [t_0, T]$$

then

$$u(t) \leq \alpha \exp\!\left(\int_{t_0}^t \beta(s)\, ds\right) \quad \text{for all } t \in [t_0, T]$$

3. Proof

Proof

Step 1. Define the function

$$v(t) = \alpha + \int_{t_0}^t \beta(s)\, u(s)\, ds$$

By hypothesis, $u(t) \leq v(t)$. Also, $v(t_0) = \alpha$ and $v$ is differentiable with

$$v'(t) = \beta(t)\, u(t)$$

by the Fundamental Theorem of Calculus.

Step 2. Since $u(t) \leq v(t)$, we get $v'(t) = \beta(t) u(t) \leq \beta(t) v(t)$. So $v$ satisfies the differential inequality

$$v'(t) - \beta(t)\, v(t) \leq 0$$

Step 3. Multiply both sides by the positive factor $e^{-\int_{t_0}^t \beta(s)\, ds}$. Define $B(t) = \int_{t_0}^t \beta(s)\, ds$, so the factor is $e^{-B(t)}$. We get

$$v'(t)\, e^{-B(t)} - \beta(t)\, v(t)\, e^{-B(t)} \leq 0$$

The left side is exactly $\frac{d}{dt}\!\left[v(t)\, e^{-B(t)}\right]$. This is because $(v e^{-B})' = v' e^{-B} + v(-\beta)e^{-B} = (v' - \beta v)e^{-B}$, using the product rule and $B'(t) = \beta(t)$.

Step 4. So $\frac{d}{dt}[v(t) e^{-B(t)}] \leq 0$ on $[t_0, T]$. The function $v(t) e^{-B(t)}$ is non-increasing. Therefore, for all $t \geq t_0$:

$$v(t)\, e^{-B(t)} \leq v(t_0)\, e^{-B(t_0)} = \alpha \cdot e^0 = \alpha$$

Step 5. Multiply both sides by $e^{B(t)} > 0$:

$$v(t) \leq \alpha\, e^{B(t)} = \alpha \exp\!\left(\int_{t_0}^t \beta(s)\, ds\right)$$

Since $u(t) \leq v(t)$, we conclude $u(t) \leq \alpha \exp\!\left(\int_{t_0}^t \beta(s)\, ds\right)$.

∎

4. The constant-coefficient case

Theorem — Gronwall's inequality (constant $\beta$)

If $\beta(s) = \beta$ is constant, the inequality simplifies to: whenever $u(t) \leq \alpha + \beta \int_{t_0}^t u(s)\, ds$, we have

$$u(t) \leq \alpha\, e^{\beta(t - t_0)}$$

This follows immediately since $\int_{t_0}^t \beta\, ds = \beta(t - t_0)$.

5. Continuous dependence on initial conditions

Consider the ODE $y' = f(t, y)$ where $f$ is Lipschitz in $y$ with constant $L$. Let $y$ and $z$ be solutions with $y(t_0) = y_0$ and $z(t_0) = z_0$ respectively (same equation, different starting points).

Theorem — Continuous dependence

$|y(t) - z(t)| \leq |y_0 - z_0|\, e^{L(t - t_0)}$ for all $t$ in the common interval of existence.

Proof

Step 1. Both solutions satisfy the integral equation:

$$y(t) = y_0 + \int_{t_0}^t f(s, y(s))\, ds, \qquad z(t) = z_0 + \int_{t_0}^t f(s, z(s))\, ds$$

Step 2. Subtract:

$$y(t) - z(t) = (y_0 - z_0) + \int_{t_0}^t [f(s, y(s)) - f(s, z(s))]\, ds$$

Step 3. Take absolute values and apply the Lipschitz condition:

$$|y(t) - z(t)| \leq |y_0 - z_0| + L \int_{t_0}^t |y(s) - z(s)|\, ds$$

Step 4. Set $u(t) = |y(t) - z(t)|$ and $\alpha = |y_0 - z_0|$. The inequality is $u(t) \leq \alpha + L \int_{t_0}^t u(s)\, ds$. Apply Gronwall with constant $\beta = L$:

$$|y(t) - z(t)| \leq |y_0 - z_0|\, e^{L(t - t_0)}$$
∎

This is a powerful result. It says that if two solutions start $\varepsilon$ apart, they stay within $\varepsilon e^{L(t-t_0)}$ of each other. The divergence is at most exponential, controlled by the Lipschitz constant.

Example. Consider $y' = \sin(y)$, which is Lipschitz with $L = 1$ (since $|\cos y| \leq 1$). If $y(0) = 0.01$ and $z(0) = 0$, then $|y(t) - z(t)| \leq 0.01 \cdot e^t$. After $t = 5$ seconds, the solutions differ by at most $0.01 \cdot e^5 \approx 1.48$. The bound is pessimistic (the actual solutions stay much closer because $\sin(y) \approx y$ near zero and the trajectories converge to zero), but it gives a guaranteed worst case.

6. Uniqueness as a corollary

Set $y_0 = z_0$ in the continuous dependence theorem. Then $\alpha = 0$ and we get $|y(t) - z(t)| \leq 0$ for all $t$. Since $|y(t) - z(t)| \geq 0$, we conclude $y(t) = z(t)$. This gives a second proof of uniqueness in Cauchy–Lipschitz — cleaner than the iterative argument.

7. Linear growth and global existence

When does a solution exist for all time, rather than blowing up at some finite $T$? A sufficient condition is linear growth.

Theorem — Global existence under linear growth

If $|f(t, y)| \leq M(1 + |y|)$ for all $(t, y)$ in the domain, and $f$ is Lipschitz in $y$, then the solution of $y' = f(t,y)$, $y(t_0) = y_0$ exists for all $t \geq t_0$.

Proof

Step 1. From the integral equation:

$$|y(t)| \leq |y_0| + \int_{t_0}^t |f(s, y(s))|\, ds \leq |y_0| + M \int_{t_0}^t (1 + |y(s)|)\, ds$$

Step 2. Rearrange: $|y(t)| \leq (|y_0| + M(t - t_0)) + M \int_{t_0}^t |y(s)|\, ds$.

Step 3. Apply Gronwall with $u(t) = |y(t)|$, $\alpha = |y_0| + M(T - t_0)$ (on any interval $[t_0, T]$), and $\beta = M$:

$$|y(t)| \leq (|y_0| + M(T - t_0))\, e^{M(t - t_0)}$$

This bound is finite for every finite $T$. So $y$ cannot blow up in finite time, and the solution extends to all of $[t_0, \infty)$.

∎

Compare with $y' = y^2$ (quadratic, not linear growth): the solution $y = 1/(1-t)$ blows up at $t = 1$. The linear growth condition rules out such behaviour.

Historical note

Thomas Hakon Gronwall (1877–1932), a Swedish-American mathematician, proved his inequality in 1919 while studying parabolic differential equations. Richard Bellman (1920–1984) proved the more general version with variable $\alpha(t)$ and $\beta(t)$ in 1943. The inequality is sometimes called the Gronwall–Bellman inequality. It has become indispensable: it appears in every ODE textbook and extends to partial differential equations, stochastic calculus, and control theory. Wherever you need to turn an integral inequality into a pointwise bound, Gronwall is the tool.

What breaks

Remove $u \geq 0$. The inequality becomes meaningless — a function that is allowed to be negative can trivially satisfy $u(t) \leq \alpha + \int \beta u\, ds$ while being arbitrarily negative (that does not violate the upper bound).

Remove continuity. The proof uses the Fundamental Theorem of Calculus to differentiate $v$. Without continuity of $u$ and $\beta$, the integral may not be differentiable, and the argument collapses. (There are generalised versions for $L^1$ functions, but the standard proof requires continuity.)

Is the exponential bound tight? Yes. The function $u(t) = \alpha e^{\beta(t - t_0)}$ satisfies $u(t) = \alpha + \beta \int_{t_0}^t u(s)\, ds$ with equality — it is the solution of $u' = \beta u$, $u(t_0) = \alpha$. So the bound cannot be improved in general.

If a non-negative function is bounded by a constant plus its own integral, it grows at most exponentially — this single inequality controls uniqueness, stability, and blow-up.
