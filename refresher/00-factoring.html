Factoring

Refresher — The prerequisite for everything

Factoring means rewriting an expression as a product. This is the reverse of expanding. It's the single most important algebraic manipulation skill — you'll use it to solve equations, simplify fractions, find restrictions on denominators, and prepare expressions for calculus. Master this before moving on.

1. Expanding (the forward direction)

Before you can factor, you need to be fluent at expanding — multiplying out brackets. Factoring is just this process in reverse.

Single bracket

$$a(b + c) = ab + ac$$

Example: $3(x + 4) = 3x + 12$. And $x(x - 5) = x^2 - 5x$.

Two brackets (FOIL)

$$(a + b)(c + d) = ac + ad + bc + bd$$

Example: $(x + 3)(x - 2) = x^2 - 2x + 3x - 6 = x^2 + x - 6$.

Perfect squares

$$(a + b)^2 = a^2 + 2ab + b^2 \qquad\qquad (a - b)^2 = a^2 - 2ab + b^2$$

Common mistake: $(a + b)^2 \neq a^2 + b^2$. The middle term $2ab$ is always there.

If you can expand confidently, you can check every factoring result by multiplying it back out.

2. Pull out common factors

Always do this first. Look for something that appears in every term — a number, a variable, or both.

$$x^2 + 6x = x(x + 6)$$

Both terms contain $x$, so pull it out. You're left with $(x + 6)$ inside.

$$3x^3 - 9x^2 = 3x^2(x - 3)$$

Both terms share $3x^2$. Factor it out.

$$4x^2 + 8x + 12 = 4(x^2 + 2x + 3)$$

All three terms share $4$.

$$6x^3y - 15x^2y^2 + 9xy = 3xy(2x^2 - 5xy + 3)$$

The greatest common factor is $3xy$. Take the largest power of each variable that appears in every term.

How to find the common factor: for each variable, take the lowest power across all terms. For the numerical coefficient, take the GCD.

4. Factor a quadratic $x^2 + bx + c$

Find two numbers $r_1$ and $r_2$ such that:

  • $r_1 + r_2 = -b$ (they add to the negative of the middle coefficient)
  • $r_1 \cdot r_2 = c$ (they multiply to the constant term)

Then $x^2 + bx + c = (x - r_1)(x - r_2)$.

Example: factor $x^2 - 5x + 6$.

Need two numbers that add to $5$ and multiply to $6$: that's $2$ and $3$.

$$x^2 - 5x + 6 = (x - 2)(x - 3)$$

Check: $(x-2)(x-3) = x^2 - 3x - 2x + 6 = x^2 - 5x + 6$. Correct.

Example: factor $x^2 + x - 12$.

Need two numbers that add to $-1$ and multiply to $-12$: that's $3$ and $-4$.

$$x^2 + x - 12 = (x - 3)(x + 4)$$

Example: factor $x^2 - 4x - 21$.

Need two numbers that add to $4$ and multiply to $-21$: that's $7$ and $-3$.

$$x^2 - 4x - 21 = (x - 7)(x + 3)$$

If you can't spot them: use the quadratic formula to find $r_1$ and $r_2$, then write $(x - r_1)(x - r_2)$. This always works, even when the roots aren't integers.

5. Factor $ax^2 + bx + c$ when $a \neq 1$

Method A (find the roots): use the quadratic formula to get $r_1$ and $r_2$, then write $a(x - r_1)(x - r_2)$. Reliable, always works.

Method B (the ac-method): find two numbers that add to $b$ and multiply to $a \cdot c$. Use them to split the middle term, then factor by grouping.

Example: factor $2x^2 + 7x + 3$.

$a \cdot c = 2 \times 3 = 6$. Need two numbers that add to $7$ and multiply to $6$: that's $1$ and $6$.

$$2x^2 + 7x + 3 = 2x^2 + x + 6x + 3 = x(2x + 1) + 3(2x + 1) = (x + 3)(2x + 1)$$

Example: factor $6x^2 - 7x - 3$.

$a \cdot c = 6 \times (-3) = -18$. Need two numbers that add to $-7$ and multiply to $-18$: that's $-9$ and $2$.

$$6x^2 - 7x - 3 = 6x^2 - 9x + 2x - 3 = 3x(2x - 3) + 1(2x - 3) = (3x + 1)(2x - 3)$$

6. Factoring by grouping (4+ terms)

Group terms in pairs and look for a common bracket.

$$x^3 + 3x^2 - x - 3 = x^2(x + 3) - 1(x + 3) = (x^2 - 1)(x + 3) = (x-1)(x+1)(x+3)$$

The last step uses $a^2 - b^2$ on $(x^2 - 1)$. Always check if you can factor further.

7. The complete checklist

  1. Common factor? Pull it out. Always first.
  2. Two terms? Check for $a^2 - b^2$, $a^3 \pm b^3$.
  3. Three terms (quadratic)? Find two numbers that add/multiply correctly. If stuck, use the quadratic formula.
  4. Four terms? Group in pairs.
  5. Can you factor further? Check each factor. $(x^2 - 4)$ is not fully factored — it's $(x-2)(x+2)$.
  6. Check by expanding. Multiply your answer back out. If you get the original expression, you're done.
