Arithmetic Rules Exercises

Refresher — 2 easy + 3 hard

Reference: Arithmetic Rules. The easy exercises check each category. The hard ones combine everything.

Easy

Exercise 1 — One from each category

Simplify each expression to a single number or simplest form:

  1. Exponents: $\;\dfrac{2^5 \cdot 2^{-3}}{4}$
  2. Logarithms: $\;\ln(e^3) + \ln(1) - 2\ln(e)$
  3. Fractions: $\;\dfrac{2}{3} + \dfrac{5}{6}$
  4. Square roots: $\;\sqrt{75} - 2\sqrt{27}$
  5. Roots & powers: $\;\left(x^{1/3}\right)^6$
Show solution

(a) $\dfrac{2^5 \cdot 2^{-3}}{4} = \dfrac{2^{5-3}}{2^2} = \dfrac{2^2}{2^2} = 1$.

(b) $\ln(e^3) + \ln(1) - 2\ln(e) = 3 + 0 - 2 = 1$.

(c) $\dfrac{2}{3} + \dfrac{5}{6} = \dfrac{4}{6} + \dfrac{5}{6} = \dfrac{9}{6} = \dfrac{3}{2}$.

(d) $\sqrt{75} = 5\sqrt{3}$ and $\sqrt{27} = 3\sqrt{3}$. So $5\sqrt{3} - 2(3\sqrt{3}) = 5\sqrt{3} - 6\sqrt{3} = -\sqrt{3}$.

(e) $(x^{1/3})^6 = x^{6/3} = x^2$.

Exercise 2 — Another from each category

Simplify:

  1. Exponents: $\;\left(\dfrac{3^4}{3^{-2}}\right)^{1/3}$
  2. Logarithms: $\;\ln\!\left(\dfrac{e^5}{\sqrt{e}}\right)$
  3. Fractions: $\;\dfrac{\;\frac{2}{x}\;}{\;\frac{4}{x^2}\;}$
  4. Square roots: $\;\dfrac{6}{\sqrt{3}}$
  5. Roots & powers: $\;\sqrt{x} \cdot \sqrt[3]{x}$
Show solution

(a) $\dfrac{3^4}{3^{-2}} = 3^{4-(-2)} = 3^6$. Then $(3^6)^{1/3} = 3^2 = 9$.

(b) $\ln\!\left(\dfrac{e^5}{\sqrt{e}}\right) = \ln(e^5) - \ln(e^{1/2}) = 5 - \dfrac{1}{2} = \dfrac{9}{2}$.

(c) Dividing fractions: $\dfrac{2}{x} \cdot \dfrac{x^2}{4} = \dfrac{2x^2}{4x} = \dfrac{x}{2}$.

(d) Rationalise: $\dfrac{6}{\sqrt{3}} = \dfrac{6\sqrt{3}}{3} = 2\sqrt{3}$.

(e) $x^{1/2} \cdot x^{1/3} = x^{1/2 + 1/3} = x^{5/6}$.

Hard

Exercise 3 — Is this correct?

A student simplifies the following expression. For each step, say whether it's correct or wrong. If wrong, give the correct result.

$$\frac{\sqrt{8} \cdot 3^2}{\sqrt{2} \cdot 9} \quad\xrightarrow{\text{step 1}}\quad \frac{\sqrt{4} \cdot 9}{\sqrt{2} \cdot 9} \quad\xrightarrow{\text{step 2}}\quad \frac{\sqrt{4}}{\sqrt{2}} \quad\xrightarrow{\text{step 3}}\quad \sqrt{4/2} \quad\xrightarrow{\text{step 4}}\quad \sqrt{2}$$
Show solution

Step 1: WRONG. $\sqrt{8} \neq \sqrt{4}$. The student confused $\sqrt{8}$ with $\sqrt{4}$. Correct: $\sqrt{8} = 2\sqrt{2}$. And $3^2 = 9$. So the numerator is $2\sqrt{2} \cdot 9 = 18\sqrt{2}$.

Let's redo it correctly from the start:

$$\frac{\sqrt{8} \cdot 3^2}{\sqrt{2} \cdot 9} = \frac{2\sqrt{2} \cdot 9}{\sqrt{2} \cdot 9} = \frac{2\sqrt{2}}{\sqrt{2}} = 2$$

The $9$'s cancel, the $\sqrt{2}$'s cancel. Answer is $2$, not $\sqrt{2}$.

Steps 2, 3, 4 are all individually valid operations ($9$ cancels, $\sqrt{a}/\sqrt{b} = \sqrt{a/b}$, $\sqrt{4/2} = \sqrt{2}$) but they're applied to a wrong starting point. The error was in step 1.

Exercise 4 — Which simplification is correct?

Simplify: $\;\;\ln\!\left(\dfrac{x^3\sqrt{x}}{e^{2x}}\right)$

Three students give three answers. Which is correct?

  • A: $\;3\ln x + \frac{1}{2}\ln x - 2x$
  • B: $\;\frac{7}{2}\ln x - 2x$
  • C: $\;\frac{3}{2}\ln x - 2x$
Show solution

Apply log rules step by step:

$$\ln\!\left(\frac{x^3\sqrt{x}}{e^{2x}}\right) = \ln(x^3\sqrt{x}) - \ln(e^{2x})$$
$$= \ln(x^3) + \ln(\sqrt{x}) - 2x$$
$$= 3\ln x + \tfrac{1}{2}\ln x - 2x$$
$$= \tfrac{7}{2}\ln x - 2x$$

A is correct (unsimplified form). B is correct (simplified form). They're the same thing.

C is wrong. The student wrote $\ln(x^3 \cdot x^{1/2})$ as $\ln(x^{3/2})$ instead of $\ln(x^{7/2})$. They multiplied the exponents instead of adding them: $x^3 \cdot x^{1/2} = x^{3 + 1/2} = x^{7/2}$, not $x^{3 \cdot 1/2}$.

Exercise 5 — Simplify this monster

Simplify to a single fraction with no roots in the denominator:

$$\frac{\;\dfrac{1}{\sqrt{x}} - \dfrac{\sqrt{x}}{x^2}\;}{1 + \dfrac{1}{x}}$$
Show solution

Step 1: convert everything to powers of $x$.

$$\frac{x^{-1/2} - x^{1/2} \cdot x^{-2}}{1 + x^{-1}} = \frac{x^{-1/2} - x^{-3/2}}{1 + x^{-1}}$$

Step 2: simplify numerator. Factor out $x^{-3/2}$ (the smaller power):

$$x^{-3/2}(x^1 - 1) = \frac{x - 1}{x^{3/2}}$$

Step 3: simplify denominator. $1 + x^{-1} = \dfrac{x + 1}{x}$.

Step 4: divide.

$$\frac{(x-1)/x^{3/2}}{(x+1)/x} = \frac{x-1}{x^{3/2}} \cdot \frac{x}{x+1} = \frac{x(x-1)}{x^{3/2}(x+1)} = \frac{x-1}{x^{1/2}(x+1)} = \frac{(x-1)\sqrt{x}}{x(x+1)}$$

The last step rationalises the denominator: multiply top and bottom by $\sqrt{x}$.

Final answer: $\;\dfrac{(x-1)\sqrt{x}}{x(x+1)}$

This exercise uses: fractional exponents, factoring out common powers, fraction division (flip and multiply), and rationalising. All from the cheat sheet — no new concepts, just fluency under pressure.
