Refresher — Complex numbers, combinatorics, elementary statistics

Three topics the course uses but never teaches from zero. This page fills the gap.

1. Complex numbers

The idea

Define $i$ as a number satisfying $i^2 = -1$. A complex number is $z = a + bi$ where $a, b \in \mathbb{R}$. We call $a = \operatorname{Re}(z)$ the real part and $b = \operatorname{Im}(z)$ the imaginary part.

Arithmetic

OperationRule
Addition$(a+bi) + (c+di)$ $= (a+c) + (b+d)i$
Multiplication$(a+bi)(c+di)$ $= (ac - bd) + (ad + bc)i$
Conjugate$\overline{a+bi}$ $= a - bi$
Modulus$|a+bi|$ $= \sqrt{a^2 + b^2}$
Division$\dfrac{z}{w}$ $= \dfrac{z\bar{w}}{|w|^2}$

Key identity: $z\bar{z} = |z|^2$. This is how you divide — multiply numerator and denominator by $\bar{w}$.

Example:

$$\frac{3+2i}{1-i} = \frac{(3+2i)(1+i)}{(1-i)(1+i)} = \frac{3+3i+2i+2i^2}{1+1} = \frac{1+5i}{2} = \frac{1}{2} + \frac{5}{2}i$$

Modulus-argument form (polar form)

Every complex number can be written as $z = r(\cos\theta + i\sin\theta)$ where $r = |z|$ and $\theta = \arg(z)$ is the angle from the positive real axis.

Euler's formula: $e^{i\theta} = \cos\theta + i\sin\theta$. So $z = re^{i\theta}$.

Multiplication in polar form: multiply moduli, add arguments.

$$z_1 z_2 = r_1 r_2 \, e^{i(\theta_1 + \theta_2)}$$

De Moivre's formula: $(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)$.

Roots of unity

The $n$-th roots of unity are the $n$ solutions to $z^n = 1$:

$$z_k = e^{2\pi i k/n}, \quad k = 0, 1, \ldots, n-1$$

They're equally spaced around the unit circle.

Solving quadratics with $\Delta < 0$

When $b^2 - 4ac < 0$, the roots are complex conjugates:

$$x = \frac{-b \pm i\sqrt{4ac - b^2}}{2a}$$

2. Combinatorics

Factorial

$n! = n \times (n-1) \times \cdots \times 2 \times 1$. By convention, $0! = 1$.

Permutations

The number of ways to arrange $k$ objects from a set of $n$ (order matters):

$$P(n, k) = \frac{n!}{(n-k)!}$$

Combinations (binomial coefficient)

The number of ways to choose $k$ objects from $n$ (order doesn't matter):

$$\binom{n}{k} = \frac{n!}{k!(n-k)!}$$

Key properties:

  • $\binom{n}{0} = \binom{n}{n} = 1$
  • $\binom{n}{k} = \binom{n}{n-k}$ (symmetry)
  • $\binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}$ (Pascal's rule)

Counting principles

  • Multiplication principle: if task A has $m$ outcomes and task B has $n$ outcomes, doing both has $m \times n$ outcomes.
  • Addition principle: if task A has $m$ outcomes and task B has $n$ outcomes (mutually exclusive), doing one or the other has $m + n$ outcomes.

3. Elementary statistics

Mean (average)

$$\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$$

Median

Sort the data. The median is the middle value (or the average of the two middle values if $n$ is even).

Variance and standard deviation

$$\text{Var}(X) = \frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2 = \overline{x^2} - \bar{x}^2$$

The second form ($\overline{x^2} - \bar{x}^2$) is the computational shortcut — "mean of squares minus square of mean."

Standard deviation: $\sigma = \sqrt{\text{Var}(X)}$.

What these become in the course

Phase 12 replaces these with rigorous definitions: the mean becomes expectation $E[X]$, variance becomes $\text{Var}(X) = E[(X - E[X])^2]$, and "data set" becomes "random variable." The formulas are the same — the conceptual framework is deeper.
