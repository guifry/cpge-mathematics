Trigonometry Exercises

Refresher — 10 exercises, easy to hard

Work each problem on paper before opening the solution. Reference: Trigonometry Refresher.

Exercise 1 — Key values

Without a calculator, evaluate: $\sin(\pi/3)$, $\cos(\pi/4)$, $\tan(\pi/6)$.

Show solution

$\sin(\pi/3) = \sqrt{3}/2$. $\;\cos(\pi/4) = \sqrt{2}/2$. $\;\tan(\pi/6) = 1/\sqrt{3} = \sqrt{3}/3$.

Exercise 2 — Other quadrants

Evaluate: $\sin(5\pi/6)$, $\cos(4\pi/3)$, $\tan(7\pi/4)$.

Show solution

$\sin(5\pi/6) = \sin(\pi - \pi/6) = \sin(\pi/6) = 1/2$. (Q2, sine positive.)

$\cos(4\pi/3) = \cos(\pi + \pi/3) = -\cos(\pi/3) = -1/2$. (Q3, cosine negative.)

$\tan(7\pi/4) = \tan(2\pi - \pi/4) = \tan(-\pi/4) = -1$. (Q4, tangent negative.)

Exercise 3 — Pythagorean identity

If $\sin\theta = 3/5$ and $\theta$ is in Q1, find $\cos\theta$ and $\tan\theta$.

Show solution

$\cos^2\theta = 1 - \sin^2\theta = 1 - 9/25 = 16/25$. Since Q1, $\cos\theta = 4/5$.

$\tan\theta = \sin\theta/\cos\theta = (3/5)/(4/5) = 3/4$.

Exercise 4 — Radians conversion

Convert $150°$ to radians. Convert $5\pi/4$ to degrees.

Show solution

$150° = 150 \times \pi/180 = 5\pi/6$.

$5\pi/4 = 5\pi/4 \times 180/\pi = 225°$.

Exercise 5 — Solve a basic trig equation

Solve for $x \in [0, 2\pi)$: $\cos(x) = -1/2$.

Show solution

$\arccos(1/2) = \pi/3$. Since we need $\cos(x) = -1/2$, the angle is in Q2 and Q3:

$x = \pi - \pi/3 = 2\pi/3$ or $x = \pi + \pi/3 = 4\pi/3$.

Solutions: $x \in \{2\pi/3, \; 4\pi/3\}$.

Exercise 6 — Double angle

If $\cos\theta = 3/5$, find $\sin(2\theta)$ and $\cos(2\theta)$.

Show solution

First: $\sin\theta = \pm\sqrt{1 - 9/25} = \pm 4/5$. Without knowing the quadrant, keep both options. Assume Q1 for this problem: $\sin\theta = 4/5$.

$\sin(2\theta) = 2\sin\theta\cos\theta = 2(4/5)(3/5) = 24/25$.

$\cos(2\theta) = \cos^2\theta - \sin^2\theta = 9/25 - 16/25 = -7/25$.

Check: $\sin^2(2\theta) + \cos^2(2\theta) = 576/625 + 49/625 = 625/625 = 1$. Correct.

Exercise 7 — Simplify using identities

Simplify: $\;\dfrac{1 - \cos(2x)}{\sin(2x)}$

Show solution

Use $1 - \cos(2x) = 2\sin^2(x)$ and $\sin(2x) = 2\sin(x)\cos(x)$:

$$\frac{2\sin^2(x)}{2\sin(x)\cos(x)} = \frac{\sin(x)}{\cos(x)} = \tan(x)$$
Exercise 8 — Quadratic in disguise

Solve for $x \in [0, 2\pi)$: $\;2\cos^2(x) + \cos(x) - 1 = 0$.

Show solution

Let $u = \cos(x)$. Then $2u^2 + u - 1 = 0$. Factor: $(2u - 1)(u + 1) = 0$.

$u = 1/2$: $\cos(x) = 1/2$, so $x = \pi/3$ or $x = 5\pi/3$.

$u = -1$: $\cos(x) = -1$, so $x = \pi$.

Solutions: $x \in \{\pi/3, \;\pi, \;5\pi/3\}$.

Exercise 9 — Addition formula

Compute $\cos(\pi/12)$ exactly. (Hint: $\pi/12 = \pi/3 - \pi/4$.)

Show solution

$\cos(\pi/3 - \pi/4) = \cos(\pi/3)\cos(\pi/4) + \sin(\pi/3)\sin(\pi/4)$

$$= \frac{1}{2}\cdot\frac{\sqrt{2}}{2} + \frac{\sqrt{3}}{2}\cdot\frac{\sqrt{2}}{2} = \frac{\sqrt{2}}{4} + \frac{\sqrt{6}}{4} = \frac{\sqrt{2} + \sqrt{6}}{4}$$

This is an exact value. $\;\cos(15°) = (\sqrt{6} + \sqrt{2})/4 \approx 0.966$.

Exercise 10 — Prove an identity

Prove that $\;\dfrac{\sin(x)}{1 + \cos(x)} = \dfrac{1 - \cos(x)}{\sin(x)}$.

Show solution

Cross-multiply: need to show $\sin(x) \cdot \sin(x) = (1 + \cos(x))(1 - \cos(x))$.

LHS: $\sin^2(x)$.

RHS: $1 - \cos^2(x) = \sin^2(x)$ (Pythagorean identity).

LHS $=$ RHS. Done.

Alternative: start from the left side, multiply numerator and denominator by $(1 - \cos x)$:

$$\frac{\sin x}{1 + \cos x} \cdot \frac{1 - \cos x}{1 - \cos x} = \frac{\sin x(1 - \cos x)}{1 - \cos^2 x} = \frac{\sin x(1 - \cos x)}{\sin^2 x} = \frac{1 - \cos x}{\sin x}$$
