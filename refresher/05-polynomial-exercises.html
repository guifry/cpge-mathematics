Polynomial Exercises

Refresher — 10 exercises, easy to hard

Work each problem on paper before opening the solution. Reference: Polynomials and Factoring.

Exercise 1 — Quadratic formula

Solve: $\;2x^2 - 7x + 3 = 0$

Show solution

$\Delta = 49 - 24 = 25$. $x = \dfrac{7 \pm 5}{4}$. So $x = 3$ or $x = 1/2$.

Quick check via Vieta: $3 + 1/2 = 7/2 = -(-7)/2$ and $3 \times 1/2 = 3/2 = 3/2$. Confirmed.

Exercise 2 — Factoring by recognition

Factor completely: $\;x^4 - 16$

Show solution

Difference of squares: $x^4 - 16 = (x^2)^2 - 4^2 = (x^2 - 4)(x^2 + 4)$.

Again: $x^2 - 4 = (x-2)(x+2)$. And $x^2 + 4$ doesn't factor over $\mathbb{R}$ (discriminant $= -16 < 0$).

Result: $(x-2)(x+2)(x^2+4)$.

Exercise 3 — Sum of cubes

Factor: $\;8x^3 + 27$

Show solution

Recognise $8x^3 = (2x)^3$ and $27 = 3^3$. Apply $a^3 + b^3 = (a+b)(a^2 - ab + b^2)$:

$$8x^3 + 27 = (2x + 3)(4x^2 - 6x + 9)$$

Check the quadratic: $\Delta = 36 - 144 = -108 < 0$. It doesn't factor further over $\mathbb{R}$.

Exercise 4 — Factoring by grouping

Factor: $\;x^3 - x^2 - 4x + 4$

Show solution

Group: $x^2(x - 1) - 4(x - 1) = (x^2 - 4)(x - 1) = (x-2)(x+2)(x-1)$.

Alternatively: test $x = 1$. $1 - 1 - 4 + 4 = 0$. So $(x-1)$ is a factor. Divide to get $x^2 - 4$, then factor.

Exercise 5 — Rational root theorem

Find all rational roots of $\;2x^3 - 3x^2 - 8x + 12 = 0$, then solve completely.

Show solution

Candidates: $\pm\{1, 2, 3, 4, 6, 12, 1/2, 3/2\}$ (divisors of 12 over divisors of 2).

Test $x = 2$: $2(8) - 3(4) - 8(2) + 12 = 16 - 12 - 16 + 12 = 0$. Root found.

Divide by $(x-2)$: $2x^3 - 3x^2 - 8x + 12 = (x-2)(2x^2 + x - 6)$.

Solve $2x^2 + x - 6 = 0$: $\Delta = 1 + 48 = 49$. $x = \dfrac{-1 \pm 7}{4}$. So $x = 3/2$ or $x = -2$.

Solutions: $x \in \{-2, \; 3/2, \; 2\}$.

Exercise 6 — Polynomial long division

Compute $\;(x^4 + 2x^3 - x - 2) \div (x + 2)$.

Show solution

Note: $x = -2$ is a root (test: $16 - 16 + 2 - 2 = 0$). So the division is exact.

Step by step:

  • $x^4 \div x = x^3$. Multiply: $x^3(x+2) = x^4 + 2x^3$. Subtract: $0x^3 - x - 2 = -x - 2$.
  • Remaining: $-x - 2$. $-x \div x = -1$. Multiply: $-1(x+2) = -x - 2$. Subtract: $0$.

Wait — we skipped terms. Let's be careful with the missing $x^2$ term. Write $x^4 + 2x^3 + 0x^2 - x - 2$.

  • $x^4 \div x = x^3$. $x^3(x+2) = x^4 + 2x^3$. Remainder: $0x^2 - x - 2$.
  • $0x^2 \div x = 0$. $0(x+2) = 0$. Remainder: $-x - 2$.
  • $-x \div x = -1$. $-1(x+2) = -x - 2$. Remainder: $0$.

Result: $x^3 - 1$. Check: $(x+2)(x^3 - 1) = x^4 + 2x^3 - x - 2$. Correct.

Exercise 7 — Substitution for disguised quadratics

Solve: $\;x^4 - 13x^2 + 36 = 0$

Show solution

Let $u = x^2$. Then $u^2 - 13u + 36 = 0$.

$\Delta = 169 - 144 = 25$. $u = \dfrac{13 \pm 5}{2}$. So $u = 9$ or $u = 4$.

$x^2 = 9 \Rightarrow x = \pm 3$. $\;x^2 = 4 \Rightarrow x = \pm 2$.

Solutions: $x \in \{-3, -2, 2, 3\}$.

Exercise 8 — Degree 3 with one known root

Given that $x = -1$ is a root of $\;x^3 + 4x^2 + x - 6 = 0$, find all roots.

Show solution

Verify: $(-1)^3 + 4(-1)^2 + (-1) - 6 = -1 + 4 - 1 - 6 = -4 \neq 0$.

Actually $x = -1$ is not a root! Let's check: $-1 + 4 - 1 - 6 = -4$. The problem statement is testing whether you verify. Let's find the actual roots.

Try $x = 1$: $1 + 4 + 1 - 6 = 0$. So $x = 1$ is a root.

Divide by $(x-1)$: $x^3 + 4x^2 + x - 6 = (x-1)(x^2 + 5x + 6) = (x-1)(x+2)(x+3)$.

Solutions: $x \in \{1, -2, -3\}$.

Lesson: always verify a "given" root before using it. Exam problems sometimes test alertness.

Exercise 9 — Vieta's formulas

Without solving the equation $x^2 - 5x + 3 = 0$, compute $r_1^2 + r_2^2$ and $1/r_1 + 1/r_2$, where $r_1, r_2$ are the roots.

Show solution

By Vieta: $r_1 + r_2 = 5$ and $r_1 r_2 = 3$.

$r_1^2 + r_2^2 = (r_1 + r_2)^2 - 2r_1 r_2 = 25 - 6 = 19$.

$\dfrac{1}{r_1} + \dfrac{1}{r_2} = \dfrac{r_1 + r_2}{r_1 r_2} = \dfrac{5}{3}$.

Vieta's formulas let you compute symmetric functions of the roots without finding the roots themselves. This technique appears constantly in CPGE algebra.

Exercise 10 — Partial fractions

Decompose into partial fractions: $\;\dfrac{3x + 5}{(x-1)(x+2)}$

Show solution

Write $\dfrac{3x+5}{(x-1)(x+2)} = \dfrac{A}{x-1} + \dfrac{B}{x+2}$.

Multiply both sides by $(x-1)(x+2)$: $3x + 5 = A(x+2) + B(x-1)$.

Cover-up method: set $x = 1$: $8 = 3A$, so $A = 8/3$. Set $x = -2$: $-1 = -3B$, so $B = 1/3$.

$$\frac{3x+5}{(x-1)(x+2)} = \frac{8/3}{x-1} + \frac{1/3}{x+2} = \frac{8}{3(x-1)} + \frac{1}{3(x+2)}$$

Partial fractions are essential for integration (Phase 3, page 9). Master this technique — it comes up on nearly every calculus exam.
