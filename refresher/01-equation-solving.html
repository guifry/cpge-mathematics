Equation Solving Toolkit

Refresher — Pen-and-paper fluency

This is a reference page. No proofs, no history — just the rules and worked examples. Everything here should be reflex-level before you start the course.

1. The golden rule

Whatever you do to one side of an equation, do the same to the other. That's it. Every technique below is a special case of this.

2. Isolating $x$ — by position

$x$ in a sum

Move everything else to the other side by adding/subtracting.

$$3x + 7 = 22 \quad\Longrightarrow\quad 3x = 15 \quad\Longrightarrow\quad x = 5$$

$x$ in a product

Divide both sides by the coefficient. Watch for zero — you cannot divide by zero.

$$5x = 30 \quad\Longrightarrow\quad x = 6$$

$x$ in a denominator

Multiply both sides by the denominator. Always state the restrictions first — find every value of $x$ that makes a denominator zero, and exclude them before you start solving.

$$\frac{3}{x-1} = 6 \quad\Longrightarrow\quad 3 = 6(x-1) \quad\Longrightarrow\quad x = \frac{3}{2} \qquad (x \neq 1)$$

Multiple restrictions: when the denominator is a product, each factor gives a restriction.

$$\frac{5}{(x-1)(x+2)} = 3$$

Restrictions: $x \neq 1$ and $x \neq -2$. Only then multiply both sides by $(x-1)(x+2)$:

$$5 = 3(x-1)(x+2) = 3(x^2+x-2) \quad\Longrightarrow\quad 3x^2 + 3x - 11 = 0$$

Solve with the quadratic formula. If either solution equals $1$ or $-2$, reject it.

The same applies to sums of fractions — factor every denominator first:

$$\frac{1}{x} + \frac{2}{x^2-4} = 0 \qquad\text{means}\qquad \frac{1}{x} + \frac{2}{(x-2)(x+2)} = 0$$

Three restrictions: $x \neq 0$, $x \neq 2$, $x \neq -2$. Write them down before you touch the equation. This habit prevents false solutions from slipping through.

$x$ inside an absolute value

Split into two cases: the expression inside is positive or negative.

$$|2x - 5| = 3 \quad\Longrightarrow\quad 2x - 5 = 3 \;\text{ or }\; 2x - 5 = -3 \quad\Longrightarrow\quad x = 4 \;\text{ or }\; x = 1$$

$x$ under a square root

Square both sides. Always check — squaring can introduce false solutions.

$$\sqrt{x+3} = 5 \quad\Longrightarrow\quad x+3 = 25 \quad\Longrightarrow\quad x = 22$$

Check: $\sqrt{22+3} = \sqrt{25} = 5$. Valid.

$x$ in an exponent

Take the logarithm of both sides.

$$2^x = 16 \quad\Longrightarrow\quad x\ln 2 = \ln 16 \quad\Longrightarrow\quad x = \frac{\ln 16}{\ln 2} = 4$$

$x$ inside a logarithm

Exponentiate both sides.

$$\ln(x) = 3 \quad\Longrightarrow\quad x = e^3$$

$x$ inside a trigonometric function

Apply the inverse trig function. Remember: multiple solutions (periodicity).

$$\sin(x) = \tfrac{1}{2} \quad\Longrightarrow\quad x = \frac{\pi}{6} + 2k\pi \;\text{ or }\; x = \frac{5\pi}{6} + 2k\pi, \quad k \in \mathbb{Z}$$

3. Exponent rules

These must be instant recall. Let $a, b > 0$ and $m, n \in \mathbb{R}$.

RuleExample
$a^m \cdot a^n = a^{m+n}$$2^3 \cdot 2^4 = 2^7 = 128$
$a^m / a^n = a^{m-n}$$5^6 / 5^2 = 5^4 = 625$
$(a^m)^n = a^{mn}$$(3^2)^4 = 3^8 = 6561$
$(ab)^n = a^n b^n$$(2 \cdot 3)^3 = 2^3 \cdot 3^3 = 216$
$a^0 = 1$$7^0 = 1$
$a^{-n} = 1/a^n$$2^{-3} = 1/8$
$a^{1/n} = \sqrt[n]{a}$$8^{1/3} = 2$

4. Logarithm rules

The logarithm $\ln$ is the inverse of $e^x$. So $\ln(e^x) = x$ and $e^{\ln x} = x$ for $x > 0$.

RuleExample
$\ln(ab) = \ln a + \ln b$$\ln(6) = \ln(2) + \ln(3)$
$\ln(a/b) = \ln a - \ln b$$\ln(5/3) = \ln 5 - \ln 3$
$\ln(a^n) = n\ln a$$\ln(8) = 3\ln(2)$
$\ln(1) = 0$
$\ln(e) = 1$
$\log_b(a) = \ln a / \ln b$$\log_2(8) = \ln 8 / \ln 2 = 3$

Domain: $\ln(x)$ is only defined for $x > 0$. Always check.

6. Square root rules

Basic identities

For $a, b \geq 0$:

  • $\sqrt{ab} = \sqrt{a}\,\sqrt{b}$
  • $\sqrt{a/b} = \sqrt{a}/\sqrt{b}$ (for $b > 0$)
  • $(\sqrt{a})^2 = a$
  • $\sqrt{a^2} = |a|$ — not $a$, because $a$ might be negative

Simplifying square roots

Factor out the largest perfect square:

$$\sqrt{48} = \sqrt{16 \cdot 3} = 4\sqrt{3} \qquad\qquad \sqrt{72} = \sqrt{36 \cdot 2} = 6\sqrt{2} \qquad\qquad \sqrt{200} = \sqrt{100 \cdot 2} = 10\sqrt{2}$$

Addition and subtraction

You can only combine square roots with the same radicand (the number under the root). Treat $\sqrt{n}$ like a variable:

$$3\sqrt{5} + 7\sqrt{5} = 10\sqrt{5} \qquad\qquad 2\sqrt{3} - 5\sqrt{3} = -3\sqrt{3}$$

You cannot add different roots: $\sqrt{2} + \sqrt{3} \neq \sqrt{5}$. There is no simplification — leave it as is.

Sometimes you need to simplify first to combine: $\sqrt{12} + \sqrt{27} = 2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$.

Multiplication

Roots multiply freely:

$$\sqrt{3} \cdot \sqrt{7} = \sqrt{21} \qquad\qquad 2\sqrt{5} \cdot 3\sqrt{2} = 6\sqrt{10}$$

Expanding products with roots works like any binomial:

$$(1 + \sqrt{3})(2 - \sqrt{3}) = 2 - \sqrt{3} + 2\sqrt{3} - 3 = -1 + \sqrt{3}$$

Conjugate pairs kill the root: $(a + \sqrt{b})(a - \sqrt{b}) = a^2 - b$.

Rationalising the denominator

Multiply top and bottom by the conjugate to remove roots from the denominator:

$$\frac{1}{\sqrt{a}} = \frac{\sqrt{a}}{a} \qquad\qquad \frac{1}{\sqrt{a} + \sqrt{b}} = \frac{\sqrt{a} - \sqrt{b}}{a - b}$$

Worked example:

$$\frac{4}{3 + \sqrt{5}} = \frac{4(3 - \sqrt{5})}{(3+\sqrt{5})(3-\sqrt{5})} = \frac{4(3 - \sqrt{5})}{9 - 5} = \frac{4(3-\sqrt{5})}{4} = 3 - \sqrt{5}$$

Roots and powers

The key identity: $\sqrt[n]{a} = a^{1/n}$. This means every root is just a fractional exponent, and all exponent rules apply.

ExpressionAs a powerSimplified
$\sqrt{x^3}$$(x^3)^{1/2}$$x^{3/2}$
$(\sqrt{x})^3$$(x^{1/2})^3$$x^{3/2}$ (same thing)
$\sqrt{x} \cdot x^2$$x^{1/2} \cdot x^2$$x^{5/2}$
$\dfrac{x^3}{\sqrt{x}}$$x^3 \cdot x^{-1/2}$$x^{5/2}$
$\sqrt[3]{x^2}$$(x^2)^{1/3}$$x^{2/3}$
$\sqrt{\sqrt{x}}$$(x^{1/2})^{1/2}$$x^{1/4}$

The rule of thumb: convert every root to a fractional exponent, then use $a^m \cdot a^n = a^{m+n}$ and $(a^m)^n = a^{mn}$. Convert back to root notation at the end if needed.

Worked example: simplify $\dfrac{\sqrt{x^5}}{\sqrt[3]{x^2}}$.

$$\frac{x^{5/2}}{x^{2/3}} = x^{5/2 - 2/3} = x^{15/6 - 4/6} = x^{11/6} = \sqrt[6]{x^{11}}$$

7. Factoring

Factoring means rewriting an expression as a product. This is the reverse of expanding. It's the single most important algebraic manipulation skill — you'll use it to solve equations, simplify fractions, and prepare expressions for calculus.

Step 1: Pull out common factors

Always do this first. Look for something that appears in every term.

$$x^2 + 6x = x(x + 6)$$

Both terms contain $x$, so pull it out. You're left with $(x + 6)$ inside.

More examples:

$$3x^3 - 9x^2 = 3x^2(x - 3) \qquad\qquad 4x^2 + 8x + 12 = 4(x^2 + 2x + 3)$$

Step 2: Factor a quadratic $x^2 + bx + c$

Find two numbers $r_1$ and $r_2$ such that:

  • $r_1 + r_2 = -b$ (they add to the negative of the middle coefficient)
  • $r_1 \cdot r_2 = c$ (they multiply to the constant term)

Then $x^2 + bx + c = (x - r_1)(x - r_2)$.

Example: factor $x^2 - 5x + 6$.

Need two numbers that add to $5$ and multiply to $6$: that's $2$ and $3$.

$$x^2 - 5x + 6 = (x - 2)(x - 3)$$

Example: factor $x^2 + x - 12$.

Need two numbers that add to $-1$ and multiply to $-12$: that's $3$ and $-4$.

$$x^2 + x - 12 = (x - 3)(x + 4)$$

If you can't spot them: use the quadratic formula to find $r_1$ and $r_2$, then write $(x - r_1)(x - r_2)$. This always works.

Step 3: Factor $ax^2 + bx + c$ when $a \neq 1$

Method A (roots): use the quadratic formula to get $r_1$ and $r_2$, then write $a(x - r_1)(x - r_2)$.

Method B (ac-method): find two numbers that add to $b$ and multiply to $a \cdot c$. Split the middle term, then group.

Example: factor $2x^2 + 7x + 3$.

$a \cdot c = 6$. Need two numbers that add to $7$ and multiply to $6$: that's $1$ and $6$.

$$2x^2 + 7x + 3 = 2x^2 + x + 6x + 3 = x(2x + 1) + 3(2x + 1) = (x + 3)(2x + 1)$$

Recognise these patterns on sight

PatternFactorisation
$a^2 - b^2$$(a-b)(a+b)$
$a^2 + 2ab + b^2$$(a+b)^2$
$a^2 - 2ab + b^2$$(a-b)^2$
$a^3 - b^3$$(a-b)(a^2 + ab + b^2)$
$a^3 + b^3$$(a+b)(a^2 - ab + b^2)$

Factoring by grouping (4 terms)

$$x^3 + 3x^2 - x - 3 = x^2(x+3) - 1(x+3) = (x^2 - 1)(x+3) = (x-1)(x+1)(x+3)$$

The checklist

  1. Common factor? Pull it out.
  2. Two terms? Check for $a^2 - b^2$, $a^3 \pm b^3$.
  3. Three terms? Find two numbers (add/multiply), or use the quadratic formula.
  4. Four terms? Group in pairs.

8. Inequalities

Same as equations, except:

  • Multiplying/dividing by a negative number flips the inequality sign.
  • $a < b$ and $c < 0 \;\Longrightarrow\; ac > bc$.

Other useful facts:

  • $a^2 \geq 0$ for all real $a$
  • $|a + b| \leq |a| + |b|$ (triangle inequality)
  • If $a, b > 0$: $\;\dfrac{a+b}{2} \geq \sqrt{ab}$ (AM-GM)

9. Systems of two equations

Substitution: solve one equation for one variable, plug into the other.

Elimination: multiply equations to match a coefficient, then subtract.

Example (elimination):

$$\begin{cases} 2x + 3y = 12 \\ 4x - y = 5 \end{cases}$$

Multiply the second by 3: $12x - 3y = 15$. Add to the first: $14x = 27$, so $x = 27/14$. Back-substitute for $y$.
