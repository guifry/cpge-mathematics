Factoring Exercises

Refresher — 10 exercises, easy to hard

Work each problem on paper before opening the solution. Reference: Factoring.

Exercise 1 — Common factor

Factor: $\;6x^2 + 18x$

Show solution

Both terms share $6x$. Pull it out: $6x(x + 3)$.

Check: $6x \cdot x + 6x \cdot 3 = 6x^2 + 18x$. Correct.

Exercise 2 — Difference of squares

Factor: $\;25x^2 - 49$

Show solution

Recognise $(5x)^2 - 7^2$. Apply $a^2 - b^2 = (a-b)(a+b)$:

$$25x^2 - 49 = (5x - 7)(5x + 7)$$
Exercise 3 — Simple quadratic

Factor: $\;x^2 + 7x + 12$

Show solution

Need two numbers that add to $-7$ and multiply to $12$: that's $-3$ and $-4$.

$$x^2 + 7x + 12 = (x + 3)(x + 4)$$

Check: $(x+3)(x+4) = x^2 + 4x + 3x + 12 = x^2 + 7x + 12$. Correct.

Exercise 4 — Quadratic with negative constant

Factor: $\;x^2 - 2x - 15$

Show solution

Need two numbers that add to $2$ and multiply to $-15$: that's $5$ and $-3$.

$$x^2 - 2x - 15 = (x - 5)(x + 3)$$

Check: $(x-5)(x+3) = x^2 + 3x - 5x - 15 = x^2 - 2x - 15$. Correct.

Exercise 5 — Perfect square trinomial

Factor: $\;9x^2 - 30x + 25$

Show solution

Recognise $a^2 - 2ab + b^2$ with $a = 3x$ and $b = 5$: check $2ab = 2(3x)(5) = 30x$. Yes.

$$9x^2 - 30x + 25 = (3x - 5)^2$$
Exercise 6 — Common factor then quadratic

Factor completely: $\;3x^3 - 12x^2 - 36x$

Show solution

Step 1: common factor $3x$. $3x(x^2 - 4x - 12)$.

Step 2: factor the quadratic. Need two numbers that add to $4$ and multiply to $-12$: that's $6$ and $-2$.

$$3x^3 - 12x^2 - 36x = 3x(x - 6)(x + 2)$$
Exercise 7 — Leading coefficient $\neq 1$

Factor: $\;3x^2 + 11x + 6$

Show solution

$a \cdot c = 3 \times 6 = 18$. Need two numbers that add to $11$ and multiply to $18$: that's $9$ and $2$.

Split the middle term: $3x^2 + 9x + 2x + 6 = 3x(x + 3) + 2(x + 3) = (3x + 2)(x + 3)$.

Check: $(3x+2)(x+3) = 3x^2 + 9x + 2x + 6 = 3x^2 + 11x + 6$. Correct.

Exercise 8 — Grouping

Factor: $\;x^3 - 2x^2 - 9x + 18$

Show solution

Group: $x^2(x - 2) - 9(x - 2) = (x^2 - 9)(x - 2)$.

Factor further: $x^2 - 9 = (x-3)(x+3)$.

$$x^3 - 2x^2 - 9x + 18 = (x - 3)(x + 3)(x - 2)$$
Exercise 9 — Nested pattern

Factor completely: $\;x^4 - 81$

Show solution

$x^4 - 81 = (x^2)^2 - 9^2 = (x^2 - 9)(x^2 + 9)$.

Factor further: $x^2 - 9 = (x-3)(x+3)$. And $x^2 + 9$ doesn't factor over $\mathbb{R}$ ($\Delta = -36 < 0$).

$$x^4 - 81 = (x - 3)(x + 3)(x^2 + 9)$$

Lesson: always check if each factor can be factored again. Stop when you reach irreducible factors.

Exercise 10 — Everything combined

Factor completely: $\;2x^4 + 4x^3 - 30x^2$

Show solution

Step 1: common factor $2x^2$. $2x^2(x^2 + 2x - 15)$.

Step 2: factor the quadratic. Need two numbers that add to $-2$ and multiply to $-15$: that's $-5$ and $3$. Wait — $(-5) + 3 = -2$ and $(-5)(3) = -15$. Yes.

$$2x^4 + 4x^3 - 30x^2 = 2x^2(x - 3)(x + 5)$$

Check: $2x^2 \cdot x = 2x^3$... actually let's verify: $(x-3)(x+5) = x^2 + 2x - 15$. Then $2x^2(x^2 + 2x - 15) = 2x^4 + 4x^3 - 30x^2$. Correct.
