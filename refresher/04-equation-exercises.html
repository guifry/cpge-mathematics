Equation Solving Exercises

Refresher — 10 exercises, easy to hard

Exercise 1 — Linear

Solve for $x$: $\;5x - 3 = 2x + 9$

Show solution

Move $x$-terms to one side: $5x - 2x = 9 + 3$, so $3x = 12$, giving $x = 4$.

Exercise 2 — Fractions

Solve for $x$: $\;\dfrac{2}{x+1} + \dfrac{3}{x-1} = 1$

Show solution

Restrictions: $x \neq 1$ and $x \neq -1$. Multiply both sides by $(x+1)(x-1)$:

$$2(x-1) + 3(x+1) = (x+1)(x-1)$$
$$2x - 2 + 3x + 3 = x^2 - 1$$
$$5x + 1 = x^2 - 1$$
$$x^2 - 5x - 2 = 0$$

Discriminant: $\Delta = 25 + 8 = 33$. Solutions: $x = \dfrac{5 \pm \sqrt{33}}{2}$. Neither equals $\pm 1$, so both are valid.

Exercise 3 — Absolute value

Solve for $x$: $\;|3x - 7| = 2x + 1$

Show solution

The right side must be $\geq 0$, so $x \geq -1/2$.

Case 1: $3x - 7 \geq 0$ (i.e. $x \geq 7/3$). Then $3x - 7 = 2x + 1$, so $x = 8$. Check: $|24-7| = 17$ and $2(8)+1 = 17$. Valid.

Case 2: $3x - 7 < 0$ (i.e. $x < 7/3$). Then $-(3x-7) = 2x+1$, so $7-3x = 2x+1$, giving $5x = 6$, $x = 6/5$. Check: $|18/5 - 7| = |{-17/5}| = 17/5$ and $12/5 + 1 = 17/5$. Valid.

Solutions: $x = 8$ and $x = 6/5$.

Exercise 4 — Square root

Solve for $x$: $\;\sqrt{2x+3} = x - 1$

Show solution

Restrictions: $2x + 3 \geq 0$ (so $x \geq -3/2$) and $x - 1 \geq 0$ (so $x \geq 1$, since the left side is non-negative).

Square both sides: $2x + 3 = x^2 - 2x + 1$, so $x^2 - 4x - 2 = 0$.

$\Delta = 16 + 8 = 24$. $x = \dfrac{4 \pm 2\sqrt{6}}{2} = 2 \pm \sqrt{6}$.

$2 - \sqrt{6} \approx -0.45 < 1$, rejected. $2 + \sqrt{6} \approx 4.45 \geq 1$, valid.

Check: $\sqrt{2(2+\sqrt{6})+3} = \sqrt{7+2\sqrt{6}} = \sqrt{(\sqrt{6}+1)^2} = \sqrt{6}+1 = (2+\sqrt{6})-1$. Confirmed.

Solution: $x = 2 + \sqrt{6}$.

Exercise 5 — Exponential

Solve for $x$: $\;3^{2x-1} = 27^x \cdot 9$

Show solution

Write everything in base 3: $27 = 3^3$ and $9 = 3^2$. So $3^{2x-1} = 3^{3x} \cdot 3^2 = 3^{3x+2}$.

Exponents must match: $2x - 1 = 3x + 2$, so $x = -3$.

Exercise 6 — Logarithm

Solve for $x$: $\;\ln(x+2) + \ln(x-1) = \ln(10)$

Show solution

Domain: $x + 2 > 0$ and $x - 1 > 0$, so $x > 1$.

Combine: $\ln((x+2)(x-1)) = \ln(10)$. Exponentiate: $(x+2)(x-1) = 10$.

$x^2 + x - 2 = 10$, so $x^2 + x - 12 = 0$. Factor: $(x+4)(x-3) = 0$.

$x = -4$ violates $x > 1$. $x = 3$: check $\ln(5) + \ln(2) = \ln(10)$. Valid.

Solution: $x = 3$.

Exercise 7 — Exponent and logarithm combined

Solve for $x$: $\;2^x + 2^{-x} = 5/2$

Show solution

Let $u = 2^x$ (so $u > 0$ and $2^{-x} = 1/u$). The equation becomes $u + 1/u = 5/2$.

Multiply by $u$: $u^2 + 1 = 5u/2$, so $2u^2 - 5u + 2 = 0$.

$\Delta = 25 - 16 = 9$. $u = \dfrac{5 \pm 3}{4}$. So $u = 2$ or $u = 1/2$.

$2^x = 2 \Rightarrow x = 1$. $\;2^x = 1/2 \Rightarrow x = -1$.

Solutions: $x = 1$ and $x = -1$.

Exercise 8 — System of equations

Solve the system:

$$\begin{cases} 3x - 2y = 7 \\ x^2 + y^2 = 25 \end{cases}$$
Show solution

From the first equation: $x = (7 + 2y)/3$. Substitute into the second:

$$\frac{(7+2y)^2}{9} + y^2 = 25$$
$$49 + 28y + 4y^2 + 9y^2 = 225$$
$$13y^2 + 28y - 176 = 0$$

$\Delta = 784 + 4(13)(176) = 784 + 9152 = 9936 = 16 \times 621$. Hmm — let's check: $\sqrt{9936} = 4\sqrt{621}$. This doesn't simplify nicely, suggesting a computation check.

Actually, let's try integer solutions on the circle $x^2 + y^2 = 25$: the points are $(\pm 3, \pm 4)$, $(\pm 4, \pm 3)$, $(\pm 5, 0)$, $(0, \pm 5)$.

Test $3x - 2y = 7$: $(3, 4) \to 9-8=1$ no. $(4, 3) \to 12-6=6$ no. $(5, 0) \to 15$ no. $(-3, -4) \to -9+8=-1$ no. $(-4, 3) \to -12-6=-18$ no. $(3, -4) \to 9+8=17$ no. $(4, -3) \to 12+6=18$ no. $(-3, 4) \to -9-8=-17$ no.

No integer solutions, so the solutions are indeed irrational: $y = \dfrac{-28 \pm 4\sqrt{621}}{26} = \dfrac{-14 \pm 2\sqrt{621}}{13}$, with $x = (7+2y)/3$.

The lesson: not every textbook-looking system has clean answers. The technique (substitution from linear into quadratic) is always the same.

Exercise 9 — Inequality

Solve: $\;\dfrac{x+3}{x-2} \leq 1$

Show solution

Never cross-multiply when you don't know the sign of the denominator. Instead, move everything to one side:

$$\frac{x+3}{x-2} - 1 \leq 0 \quad\Longrightarrow\quad \frac{x+3 - (x-2)}{x-2} \leq 0 \quad\Longrightarrow\quad \frac{5}{x-2} \leq 0$$

The numerator $5 > 0$ always. So we need $x - 2 < 0$, i.e. $x < 2$. (Not $x \leq 2$ because $x = 2$ makes the denominator zero.)

Solution: $x \in (-\infty, 2)$.

Exercise 10 — Trigonometric equation

Solve for $x \in [0, 2\pi)$: $\;2\sin^2(x) - 3\sin(x) + 1 = 0$

Show solution

Let $u = \sin(x)$. Then $2u^2 - 3u + 1 = 0$. Factor: $(2u - 1)(u - 1) = 0$.

$u = 1/2$: $\sin(x) = 1/2$, so $x = \pi/6$ or $x = 5\pi/6$.

$u = 1$: $\sin(x) = 1$, so $x = \pi/2$.

Solutions: $x \in \{\pi/6, \;\pi/2, \;5\pi/6\}$.
