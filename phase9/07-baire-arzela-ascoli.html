Baire Category and Arzelà–Ascoli

Phase 9 — Topology of Metric Spaces, Page 7

The problem

Two deep questions about "size" and "compactness" in infinite-dimensional spaces.

First: can $\mathbb{R}$ be written as a countable union of "thin" sets? Intuitively, $\mathbb{R}$ is too large for this. Baire (1899) made "thin" precise and proved the intuition correct: a complete metric space cannot be decomposed into countably many nowhere dense pieces.

Second: in $\mathbb{R}^n$, Heine–Borel (page 4) tells us exactly which sets are compact: the closed and bounded ones. But in $C([0, 1])$ with the sup metric, the closed unit ball $\{f : \|f\|_\infty \leq 1\}$ is closed and bounded yet not compact. What extra condition makes a family of functions compact?

1. Nowhere dense sets

Definition — Nowhere dense

A subset $A$ of a metric space $X$ is nowhere dense if its closure $\overline{A}$ has empty interior: $\mathrm{int}(\overline{A}) = \varnothing$.

Equivalently, $A$ is nowhere dense if every open ball contains an open sub-ball that misses $\overline{A}$. The set is "thin" in the sense that it is dense in no region of the space, no matter how small.

Examples. $\mathbb{Z}$ in $\mathbb{R}$ is nowhere dense: $\overline{\mathbb{Z}} = \mathbb{Z}$, which contains no interval. The set $\{0\} \cup \{1/n : n \geq 1\}$ is nowhere dense: its closure is itself, and it contains no interval. The Cantor set is nowhere dense (its construction removes all intervals), yet it is uncountable — nowhere dense does not mean small in the cardinality sense.

2. Meagre and comeagre sets

Definition — Meagre and comeagre

A set $A \subseteq X$ is meagre (or of the first category) if it is a countable union of nowhere dense sets: $A = \bigcup_{n=1}^{\infty} A_n$ with each $A_n$ nowhere dense.

A set is comeagre (or residual) if its complement is meagre.

Meagre sets are "topologically negligible" — they are the topological analogue of measure-zero sets. A comeagre set contains "most" points in the topological sense. Baire's theorem says that in a complete metric space, comeagre sets are genuinely large.

3. The Baire category theorem

Theorem — Baire category theorem

Let $(X, d)$ be a complete metric space. Then:

  1. $X$ is not meagre (it is of the second category).
  2. Every comeagre set is dense.
  3. A countable intersection of open dense sets is dense.

These three statements are equivalent.

Proof

We prove (3), which is the most useful formulation. Let $G_1, G_2, G_3, \ldots$ be open dense subsets of $X$. We must show $\bigcap_{n=1}^{\infty} G_n$ is dense, i.e. it meets every non-empty open set.

Let $U_0$ be a non-empty open set. Since $G_1$ is dense, $U_0 \cap G_1 \neq \varnothing$. This intersection is open (both sets are open), so it contains some closed ball $\overline{B}(x_1, r_1)$ with $r_1 \leq 1$.

Since $G_2$ is dense, $B(x_1, r_1) \cap G_2 \neq \varnothing$. Choose $\overline{B}(x_2, r_2) \subseteq B(x_1, r_1) \cap G_2$ with $r_2 \leq 1/2$.

Inductively, we obtain nested closed balls $\overline{B}(x_n, r_n) \subseteq B(x_{n-1}, r_{n-1}) \cap G_n$ with $r_n \leq 1/n$. The centres form a Cauchy sequence: for $m > n$, $x_m \in \overline{B}(x_n, r_n)$, so $d(x_m, x_n) \leq r_n \leq 1/n \to 0$.

Since $X$ is complete, $x_n \to x^*$ for some $x^* \in X$. For each $n$, the tail $x_m$ ($m \geq n$) lies in the closed ball $\overline{B}(x_n, r_n)$, so $x^*$ does too. Hence $x^* \in \overline{B}(x_n, r_n) \subseteq G_n$ for every $n$, and $x^* \in \overline{B}(x_1, r_1) \subseteq U_0$. Therefore $x^* \in U_0 \cap \bigcap_{n=1}^{\infty} G_n$.

∎

The proof is a nested intervals argument in disguise, generalised from $\mathbb{R}$ to arbitrary complete metric spaces. The radii $r_n \to 0$ force the centres to converge; completeness ensures the limit exists; closedness of each ball traps the limit inside.

4. Application: generic continuous functions are nowhere differentiable

Weierstrass (1872) shocked the mathematical world by exhibiting a continuous function that is differentiable at no point. Baire's theorem shows this is not pathological but typical.

For each $n \geq 1$ and $x \in [0, 1]$, define

$$D_n = \left\{f \in C([0,1]) : \exists\, x \in [0,1],\, \sup_{0 < |h| < 1/n} \left|\frac{f(x+h) - f(x)}{h}\right| \leq n \right\}$$

Each $D_n$ consists of functions that have a "bounded difference quotient" of size at most $n$ at some point, with resolution $1/n$. One can show each $D_n$ is closed and nowhere dense in $(C([0,1]), \|\cdot\|_\infty)$. The set of functions differentiable at some point is contained in $\bigcup_{n=1}^{\infty} D_n$, which is meagre. By Baire, the comeagre set — functions differentiable at no point — is dense. Weierstrass's monster is the norm, not the exception.

5. Equicontinuity

Definition — Equicontinuity

Let $K$ be a metric space. A family $\mathcal{F} \subseteq C(K)$ is equicontinuous if for every $\varepsilon > 0$ there exists $\delta > 0$ such that for all $f \in \mathcal{F}$ and all $x, y \in K$,

$$d(x, y) < \delta \implies |f(x) - f(y)| < \varepsilon$$

The definition looks like uniform continuity, but with an extra uniformity: the same $\delta$ works for every function in $\mathcal{F}$ and every point in $K$ simultaneously. A single function is equicontinuous (as a singleton family) if and only if it is uniformly continuous.

Example. The family $\{f_n(x) = x/n : n \geq 1\}$ on $[0, 1]$ is equicontinuous: $|f_n(x) - f_n(y)| = |x - y|/n \leq |x - y|$, so $\delta = \varepsilon$ works for all $n$. The family $\{g_n(x) = \sin(nx) : n \geq 1\}$ is not: the oscillation frequency grows, so no single $\delta$ controls all functions.

6. The Arzelà–Ascoli theorem

Theorem — Arzelà–Ascoli

Let $K$ be a compact metric space. A subset $\mathcal{F} \subseteq C(K)$ (with the sup metric) is relatively compact (i.e. $\overline{\mathcal{F}}$ is compact) if and only if $\mathcal{F}$ is bounded and equicontinuous.

Proof — Bounded + equicontinuous implies relatively compact

Let $(f_n)$ be a sequence in $\mathcal{F}$. We extract a uniformly convergent subsequence using Cantor's diagonal argument.

Since $K$ is compact, it is separable (totally bounded implies separable): let $\{q_1, q_2, q_3, \ldots\}$ be a countable dense subset.

The sequence $(f_n(q_1))_{n \geq 1}$ is bounded in $\mathbb{R}$ (since $\mathcal{F}$ is bounded), so by Bolzano–Weierstrass it has a convergent subsequence $(f_{n_k^{(1)}}(q_1))$. From this subsequence, extract a further subsequence $(f_{n_k^{(2)}})$ such that $(f_{n_k^{(2)}}(q_2))$ converges. Continue: at step $j$, extract a subsequence converging at $q_j$.

The diagonal subsequence $g_k = f_{n_k^{(k)}}$ converges at every $q_j$ (for $j$ fixed, $g_k$ is eventually a subsequence of the $j$-th extraction).

We claim $g_k$ converges uniformly on all of $K$. Fix $\varepsilon > 0$. By equicontinuity, choose $\delta > 0$ such that $d(x, y) < \delta$ implies $|f(x) - f(y)| < \varepsilon/3$ for all $f \in \mathcal{F}$. Since $\{q_j\}$ is dense and $K$ is compact, finitely many balls $B(q_{j_1}, \delta), \ldots, B(q_{j_p}, \delta)$ cover $K$.

Since $g_k$ converges at each $q_{j_i}$, there exists $N$ such that for $k, l \geq N$, $|g_k(q_{j_i}) - g_l(q_{j_i})| < \varepsilon/3$ for all $i = 1, \ldots, p$.

For any $x \in K$, choose $q_{j_i}$ with $d(x, q_{j_i}) < \delta$. Then for $k, l \geq N$:

$$|g_k(x) - g_l(x)| \leq |g_k(x) - g_k(q_{j_i})| + |g_k(q_{j_i}) - g_l(q_{j_i})| + |g_l(q_{j_i}) - g_l(x)| < \frac{\varepsilon}{3} + \frac{\varepsilon}{3} + \frac{\varepsilon}{3} = \varepsilon$$

This holds for all $x \in K$, so $\|g_k - g_l\|_\infty < \varepsilon$ for $k, l \geq N$. The subsequence $(g_k)$ is Cauchy in $(C(K), \|\cdot\|_\infty)$, which is complete (page 3), so it converges.

∎
Proof — Relatively compact implies bounded + equicontinuous

Bounded: $\overline{\mathcal{F}}$ is compact, hence totally bounded, hence bounded. So $\mathcal{F} \subseteq \overline{\mathcal{F}}$ is bounded.

Equicontinuous: Fix $\varepsilon > 0$. Since $\overline{\mathcal{F}}$ is compact (hence totally bounded), cover it by finitely many balls of radius $\varepsilon/3$ in the sup norm, centred at $g_1, \ldots, g_m$. Each $g_i$ is continuous on the compact set $K$, hence uniformly continuous (page 5, Heine's theorem): choose $\delta_i$ so that $d(x, y) < \delta_i$ implies $|g_i(x) - g_i(y)| < \varepsilon/3$. Set $\delta = \min(\delta_1, \ldots, \delta_m)$.

For any $f \in \mathcal{F}$, choose $g_i$ with $\|f - g_i\|_\infty < \varepsilon/3$. If $d(x, y) < \delta$:

$$|f(x) - f(y)| \leq |f(x) - g_i(x)| + |g_i(x) - g_i(y)| + |g_i(y) - f(y)| < \frac{\varepsilon}{3} + \frac{\varepsilon}{3} + \frac{\varepsilon}{3} = \varepsilon$$

The same $\delta$ works for all $f \in \mathcal{F}$, so $\mathcal{F}$ is equicontinuous.

∎

7. Phase 9 recap

We have completed the topology of metric spaces. Here is what we built:

  • Page 1: metric spaces — the abstraction of distance.
  • Page 2: open and closed sets — the topology a metric induces.
  • Page 3: completeness — spaces where Cauchy sequences converge.
  • Page 4: compactness — open covers, sequential compactness, Heine–Borel.
  • Page 5: continuity on compact sets — extreme values, uniform continuity, homeomorphisms.
  • Page 6: Banach fixed point theorem — contractions, iteration, existence proofs.
  • Page 7: Baire category and Arzelà–Ascoli — topological size and compactness in function spaces.

The properties that made analysis work on $\mathbb{R}$ — completeness, compactness, the extreme value theorem, uniform continuity — have been identified and abstracted. They are not accidents of the real number system but consequences of the metric structure. Any space with these properties supports the same theorems.

Historical note

René Baire proved the category theorem in his 1899 doctoral thesis, originally for $\mathbb{R}$. The extension to complete metric spaces was straightforward once the abstract framework existed. The theorem became a workhorse of functional analysis: the uniform boundedness principle, the open mapping theorem, and the closed graph theorem all rely on it.

Cesare Arzelà (1882–1883) and Giulio Ascoli (1884) independently found the compactness criterion for families of functions. Ascoli identified equicontinuity as the key condition; Arzelà proved the theorem in its modern form. Their result is the main tool for proving existence of solutions to partial differential equations: show that a sequence of approximate solutions is bounded and equicontinuous, extract a convergent subsequence, and verify the limit solves the equation.

What breaks

Without completeness, Baire fails. $\mathbb{Q} = \bigcup_{q \in \mathbb{Q}} \{q\}$ is a countable union of nowhere dense sets (each singleton $\{q\}$ is nowhere dense in $\mathbb{Q}$, since every interval contains irrationals). So $\mathbb{Q}$ is meagre in itself — it is "topologically small" in its own metric. This cannot happen in a complete space.

Without equicontinuity, Arzelà–Ascoli fails. Consider $f_n(x) = x^n$ on $[0, 1]$. The family is bounded ($\|f_n\|_\infty \leq 1$) but not equicontinuous: near $x = 1$, the functions develop an increasingly sharp transition from values near 0 to the value 1. The sequence converges pointwise to the discontinuous function $\mathbf{1}_{\{1\}}$, but has no uniformly convergent subsequence.

Without boundedness, Arzelà–Ascoli fails. The constant functions $f_n(x) = n$ are equicontinuous (each is Lipschitz with constant 0, and $\delta = 1$ works for any $\varepsilon$). But $\|f_n\|_\infty = n \to \infty$, so no subsequence converges.

Baire says a complete space is topologically large; Arzelà–Ascoli says equicontinuity is the missing ingredient for compactness in function spaces.
