Basis and Dimension

Phase 6 — Linear Algebra, Page 2

The problem

In $\mathbb{R}^3$, three vectors that don't all lie in the same plane can express any point as a combination of them. Two vectors give you only a plane. One gives a line. How many vectors do you need to describe an entire space? And does that number depend on which vectors you pick, or is it an intrinsic property of the space itself?

For $\mathbb{R}^3$ the answer feels obvious: three. But what about the space of polynomials of degree $\leq 5$? Or the solution space of a differential equation? Steinitz (1913) proved that the answer is always well-defined: every basis of a given space has the same number of elements. That number is the dimension.

1. Linear combinations and span

Let $V$ be a vector space over $\mathbb{K}$ and let $v_1, \ldots, v_n \in V$. A linear combination of these vectors is any expression of the form

$$\lambda_1 v_1 + \lambda_2 v_2 + \cdots + \lambda_n v_n$$

where $\lambda_1, \ldots, \lambda_n \in \mathbb{K}$. The span (or linear hull) of $v_1, \ldots, v_n$ is the set of all such combinations:

$$\text{Span}(v_1, \ldots, v_n) = \left\{ \sum_{i=1}^{n} \lambda_i v_i : \lambda_i \in \mathbb{K} \right\}$$

The span is always a subspace of $V$ (check the subspace criterion from page 1: it is non-empty since $0 = 0 \cdot v_1 + \cdots + 0 \cdot v_n$, and any linear combination of linear combinations is a linear combination).

If $\text{Span}(v_1, \ldots, v_n) = V$, we say $v_1, \ldots, v_n$ span $V$, or form a generating set (or spanning set) for $V$.

2. Linear independence

Definition — Linear independence

Vectors $v_1, \ldots, v_n \in V$ are linearly independent if the only way to write $0$ as a linear combination is the trivial one:

$$\lambda_1 v_1 + \cdots + \lambda_n v_n = 0 \implies \lambda_1 = \cdots = \lambda_n = 0$$

If the vectors are not linearly independent, they are linearly dependent: there exist scalars $\lambda_i$, not all zero, with $\sum \lambda_i v_i = 0$.

Equivalently, $v_1, \ldots, v_n$ are linearly dependent if and only if one of them is a linear combination of the others. To see this: if $\lambda_j \neq 0$ in a non-trivial relation, then $v_j = -\lambda_j^{-1} \sum_{i \neq j} \lambda_i v_i$.

3. Examples in $\mathbb{R}^3$

The standard basis $e_1 = (1,0,0)$, $e_2 = (0,1,0)$, $e_3 = (0,0,1)$. Suppose $\lambda_1 e_1 + \lambda_2 e_2 + \lambda_3 e_3 = 0$. This gives $(\lambda_1, \lambda_2, \lambda_3) = (0,0,0)$, so they are linearly independent. Every $(a,b,c) = ae_1 + be_2 + ce_3$, so they span $\mathbb{R}^3$.

Now consider $u = (1,0,0)$, $v = (0,1,0)$, $w = (1,1,0)$. These are linearly dependent: $w = u + v$, i.e. $1 \cdot u + 1 \cdot v + (-1) \cdot w = 0$. They all lie in the $xy$-plane, so they span only a two-dimensional subspace.

4. Basis: independence meets spanning

Definition — Basis

A basis of $V$ is a family of vectors that is both linearly independent and spanning. Equivalently, a basis is a minimal spanning set, or a maximal linearly independent set.

The crucial property of a basis is unique representation: every vector can be written as a linear combination in exactly one way.

Theorem — Unique representation

If $(e_1, \ldots, e_n)$ is a basis of $V$, then every $v \in V$ can be written uniquely as $v = \lambda_1 e_1 + \cdots + \lambda_n e_n$.

Proof

Existence. Since the $e_i$ span $V$, at least one such expression exists.

Uniqueness. Suppose $v = \lambda_1 e_1 + \cdots + \lambda_n e_n$ and $v = \mu_1 e_1 + \cdots + \mu_n e_n$. Subtracting: $(\lambda_1 - \mu_1)e_1 + \cdots + (\lambda_n - \mu_n)e_n = 0$. By linear independence, $\lambda_i - \mu_i = 0$ for all $i$, so $\lambda_i = \mu_i$.

∎

The scalars $(\lambda_1, \ldots, \lambda_n)$ are the coordinates (or components) of $v$ in the basis $(e_1, \ldots, e_n)$. A different basis gives different coordinates for the same vector — we will study this translation on page 4.

5. The Steinitz exchange lemma

This is the engine that makes dimension well-defined. It says: an independent set can never be larger than a spanning set.

Theorem — Steinitz exchange lemma

Let $v_1, \ldots, v_m$ be linearly independent in $V$, and let $w_1, \ldots, w_n$ span $V$. Then $m \leq n$, and one can replace $m$ of the $w_j$ by $v_1, \ldots, v_m$ to obtain a new spanning set.

Proof

We proceed by induction on $m$.

Base case ($m = 0$). There is nothing to prove — $0 \leq n$ and the spanning set is unchanged.

Inductive step. Suppose the result holds for $m - 1$. By the inductive hypothesis, after replacing $m - 1$ of the $w_j$, we have a spanning set $\{v_1, \ldots, v_{m-1}, w_{i_1}, \ldots, w_{i_{n-m+1}}\}$ (relabelling, call these remaining $w$'s simply $w_1, \ldots, w_r$ where $r = n - m + 1$).

Step 1. Since this set spans $V$, we can write

$$v_m = \alpha_1 v_1 + \cdots + \alpha_{m-1} v_{m-1} + \beta_1 w_1 + \cdots + \beta_r w_r$$

Step 2. At least one $\beta_j \neq 0$. If all $\beta_j = 0$, then $v_m = \alpha_1 v_1 + \cdots + \alpha_{m-1} v_{m-1}$, contradicting the linear independence of $v_1, \ldots, v_m$.

Step 3. Say $\beta_1 \neq 0$ (relabelling if necessary). Then

$$w_1 = \beta_1^{-1}\bigl(v_m - \alpha_1 v_1 - \cdots - \alpha_{m-1} v_{m-1} - \beta_2 w_2 - \cdots - \beta_r w_r\bigr)$$

so $w_1 \in \text{Span}(v_1, \ldots, v_m, w_2, \ldots, w_r)$. Therefore the set $\{v_1, \ldots, v_m, w_2, \ldots, w_r\}$ spans $V$ (anything expressible via the old spanning set is expressible via this one, since $w_1$ is recoverable).

Step 4. We replaced one $w_j$ by $v_m$, bringing the total replacements to $m$. The new spanning set has $m + (r - 1) = m + n - m = n$ elements. In particular, we needed $r \geq 1$, i.e. $n - m + 1 \geq 1$, so $m \leq n$.

∎

6. All bases have the same size

Theorem — Invariance of basis cardinality

If $V$ has a finite basis, then every basis of $V$ has the same number of elements.

Proof

Let $\mathcal{B}_1$ be a basis of size $m$ and $\mathcal{B}_2$ a basis of size $n$. Since $\mathcal{B}_1$ is independent and $\mathcal{B}_2$ spans $V$, Steinitz gives $m \leq n$. Since $\mathcal{B}_2$ is independent and $\mathcal{B}_1$ spans $V$, Steinitz gives $n \leq m$. Therefore $m = n$.

∎

7. Dimension

Definition — Dimension

The dimension of a finite-dimensional vector space $V$, written $\dim(V)$, is the number of elements in any basis of $V$. This is well-defined by the theorem above.

Examples. $\dim(\mathbb{R}^n) = n$ (standard basis has $n$ vectors). $\dim(\mathbb{R}_d[x]) = d + 1$ (the monomials $1, x, x^2, \ldots, x^d$ form a basis). $\dim(\mathcal{M}_{m,n}(\mathbb{R})) = mn$ (the matrices with a single $1$ and all other entries $0$ form a basis). $\dim(\{0\}) = 0$ (the empty set is a basis of the trivial space).

8. Existence of a basis

Theorem — Existence of basis

Every finite-dimensional vector space has a basis. More precisely: any linearly independent set can be extended to a basis, and any spanning set can be reduced to a basis.

Proof

Extending an independent set. Let $S = \{v_1, \ldots, v_k\}$ be linearly independent but not spanning. Then there exists $v_{k+1} \in V$ with $v_{k+1} \notin \text{Span}(S)$. The set $S \cup \{v_{k+1}\}$ is linearly independent (if $\lambda_1 v_1 + \cdots + \lambda_k v_k + \lambda_{k+1} v_{k+1} = 0$ with $\lambda_{k+1} \neq 0$, then $v_{k+1} \in \text{Span}(S)$, a contradiction; so $\lambda_{k+1} = 0$, and independence of $S$ forces all others to be $0$ as well).

Repeat. Each step increases $|S|$ by one. By Steinitz, $|S|$ can never exceed the size of any spanning set, so the process terminates. When it does, $S$ spans $V$ (otherwise we could add another vector), so $S$ is a basis.

Reducing a spanning set. If $S = \{w_1, \ldots, w_n\}$ spans $V$ but is dependent, some $w_j$ is a linear combination of the others. Remove it; the remaining set still spans $V$. Repeat until the set is independent. The result is a basis.

∎
Historical note

Ernst Steinitz proved the exchange lemma in his monumental 1913 paper Algebraische Theorie der Körper, which also laid the foundations of modern field theory. The concept of dimension feels obvious in $\mathbb{R}^3$ — you need three numbers to specify a point. But in abstract spaces like polynomial spaces, the fact that dimension is well-defined is a genuine theorem requiring proof. Before Steinitz, this was assumed without justification.

What breaks

Infinite-dimensional spaces. The space of all polynomials $\mathbb{R}[x]$ has no finite basis — $\{1, x, x^2, \ldots\}$ is a basis, but it is infinite. The space $C[0,1]$ of continuous functions is "even more infinite": it requires an uncountable basis (a Hamel basis), whose existence needs the axiom of choice. Finite-dimensional intuition breaks down: in infinite dimensions, a proper subspace can have the same "size" as the whole space.

Drop the field to a ring. Over $\mathbb{Z}$, free modules can have well-defined rank, but general modules need not. The invariant basis number property (every two bases have the same size) can fail for modules over certain rings.

A basis is a linearly independent spanning set; Steinitz guarantees that every basis of a given space has the same size, so dimension is an intrinsic property.
