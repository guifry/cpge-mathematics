Vector Spaces

Phase 6 — Linear Algebra, Page 1

The problem

Systems of linear equations appear everywhere — balancing chemical reactions, analysing electrical circuits, computing forces in a bridge. Gauss (1810s) developed systematic elimination for solving them. But once you solve such a system, a striking pattern emerges: the set of solutions is closed under addition and scaling.

If $x$ and $y$ both satisfy a homogeneous system $Ax = 0$, then so does $x + y$ and so does $3x$. The solutions live in some kind of "space." Grassmann (1844) asked: what is the most general structure with these two properties? His answer — the vector space — is the foundation of all linear algebra.

1. Motivating example: solutions of $Ax = 0$

Consider the system $x_1 + 2x_2 - x_3 = 0$. The solutions form a set $S \subseteq \mathbb{R}^3$. Take two solutions, say $v = (1, 0, 1)$ and $w = (-2, 1, 0)$. Then $v + w = (-1, 1, 1)$ satisfies the equation ($-1 + 2 - 1 = 0$), and $5v = (5, 0, 5)$ satisfies it too ($5 + 0 - 5 = 0$).

This is not a coincidence. If $Av = 0$ and $Aw = 0$, then $A(v + w) = Av + Aw = 0 + 0 = 0$ and $A(\lambda v) = \lambda Av = \lambda \cdot 0 = 0$. Linearity of matrix multiplication forces $S$ to be closed under addition and scalar multiplication. The zero vector $0$ is always in $S$. So $S$ has a very rigid algebraic structure. Let us axiomatise it.

2. Definition: vector space

Definition — Vector space

A vector space over a field $\mathbb{K}$ (for us, $\mathbb{K} = \mathbb{R}$ or $\mathbb{C}$) is a set $V$ equipped with two operations:

  • Addition: $V \times V \to V$, written $(u, v) \mapsto u + v$
  • Scalar multiplication: $\mathbb{K} \times V \to V$, written $(\lambda, v) \mapsto \lambda v$

satisfying the following eight axioms for all $u, v, w \in V$ and $\lambda, \mu \in \mathbb{K}$:

  1. Associativity of addition: $(u + v) + w = u + (v + w)$
  2. Commutativity of addition: $u + v = v + u$
  3. Additive identity: there exists $0_V \in V$ such that $v + 0_V = v$
  4. Additive inverse: for each $v$, there exists $-v$ such that $v + (-v) = 0_V$
  5. Compatibility of scalar multiplication: $\lambda(\mu v) = (\lambda\mu) v$
  6. Identity scalar: $1 \cdot v = v$
  7. Distributivity over vector addition: $\lambda(u + v) = \lambda u + \lambda v$
  8. Distributivity over scalar addition: $(\lambda + \mu)v = \lambda v + \mu v$

The elements of $V$ are called vectors. The elements of $\mathbb{K}$ are called scalars.

Axioms 1–4 say $(V, +)$ is an abelian group. Axioms 5–8 say scalar multiplication interacts with addition the way you would expect. The point is that these eight properties are all you need — every theorem in linear algebra follows from them alone.

A few immediate consequences worth noting. First, the zero vector is unique: if $0'$ is another additive identity, then $0' = 0' + 0 = 0$. Second, $0 \cdot v = 0_V$ for any $v$ (proof: $0 \cdot v = (0 + 0)v = 0v + 0v$ by axiom 8, so subtracting $0v$ from both sides gives $0_V = 0v$). Third, $(-1) \cdot v = -v$ (proof: $v + (-1)v = 1 \cdot v + (-1)v = (1 + (-1))v = 0v = 0_V$, so $(-1)v$ is the additive inverse of $v$). These small facts get used constantly.

3. Examples

$\mathbb{R}^n$. The set of $n$-tuples $(x_1, \ldots, x_n)$ with componentwise addition and scalar multiplication. The zero vector is $(0, \ldots, 0)$. All eight axioms are inherited from the arithmetic of $\mathbb{R}$. This is the prototypical vector space.

Polynomials of degree $\leq n$. Write $\mathbb{R}_n[x]$ for the set of all polynomials with real coefficients and degree at most $n$. Addition: add coefficients. Scalar multiplication: multiply every coefficient. The zero vector is the zero polynomial. Closure under addition holds because the degree of a sum is at most the max of the degrees. This is a vector space over $\mathbb{R}$.

Continuous functions $C[a,b]$. The set of all continuous real-valued functions on $[a,b]$. Addition: $(f + g)(x) = f(x) + g(x)$. Scalar multiplication: $(\lambda f)(x) = \lambda f(x)$. The zero vector is the constant function $0$. The sum and scalar multiple of continuous functions are continuous (proved in Phase 2, page 6), so closure holds.

$m \times n$ matrices. The set $\mathcal{M}_{m,n}(\mathbb{R})$ of all $m \times n$ real matrices, with entrywise addition and scalar multiplication. The zero vector is the zero matrix. This is really just $\mathbb{R}^{mn}$ in disguise.

4. A non-example

The set of polynomials of degree exactly $n$ is not a vector space. It fails closure under addition: take $p(x) = x^2$ and $q(x) = -x^2 + 1$. Both have degree exactly 2, but $p + q = 1$ has degree 0. Even worse, the zero polynomial has no well-defined degree, so there is no additive identity in the set.

Another non-example: the set $\{(x, y) \in \mathbb{R}^2 : x \geq 0\}$. It contains $v = (1, 0)$, but $(-1) \cdot v = (-1, 0) \notin$ the set. Not closed under scalar multiplication.

5. Subspaces

Often we care about a subset of a vector space that is itself a vector space with the same operations. We call such a subset a subspace.

Definition — Subspace

Let $V$ be a vector space over $\mathbb{K}$. A subset $W \subseteq V$ is a subspace of $V$ if $W$ is non-empty and is closed under addition and scalar multiplication: for all $u, v \in W$ and $\lambda \in \mathbb{K}$, we have $u + v \in W$ and $\lambda v \in W$.

You do not need to check all eight axioms for $W$. Associativity, commutativity, distributivity, etc., are inherited from $V$ because $W \subseteq V$. The only things that could fail are closure (does the operation land back in $W$?) and the existence of identity/inverses. The following criterion makes checking even easier.

6. The subspace criterion

Theorem — Subspace criterion

Let $V$ be a vector space over $\mathbb{K}$ and $W \subseteq V$. Then $W$ is a subspace of $V$ if and only if:

  1. $W \neq \emptyset$
  2. For all $u, v \in W$ and $\lambda, \mu \in \mathbb{K}$: $\lambda u + \mu v \in W$
Proof

Step 1. ($\Rightarrow$) If $W$ is a subspace, it contains $0_V$ (since $W \neq \emptyset$, pick $w \in W$; then $0 \cdot w = 0_V \in W$). Closure under addition and scalar multiplication give $\lambda u + \mu v \in W$.

Step 2. ($\Leftarrow$) Suppose conditions 1 and 2 hold. Since $W \neq \emptyset$, pick any $w \in W$. Setting $\lambda = 0, \mu = 0$ gives $0 \cdot w + 0 \cdot w = 0_V \in W$, so $W$ contains the zero vector.

Step 3. For any $v \in W$, set $\lambda = -1, \mu = 0$: $(-1)v + 0 \cdot v = -v \in W$. So additive inverses exist in $W$.

Step 4. Closure under addition: set $\lambda = \mu = 1$, giving $u + v \in W$. Closure under scalar multiplication: set $\mu = 0$, giving $\lambda u \in W$. All remaining axioms (associativity, commutativity, distributivity) are inherited from $V$. So $W$ is a vector space under the same operations.

∎

7. Examples of subspaces

Lines through the origin in $\mathbb{R}^3$. A line through the origin has the form $W = \{\lambda v : \lambda \in \mathbb{R}\}$ for some fixed non-zero $v$. This is a subspace: $\lambda(\alpha v) + \mu(\beta v) = (\lambda\alpha + \mu\beta) v \in W$.

Planes through the origin in $\mathbb{R}^3$. A plane through the origin is the solution set of a single equation $ax + by + cz = 0$. This is exactly the kernel of the matrix $A = (a, b, c)$, and we showed in Section 1 that kernels are subspaces.

The kernel of any matrix. More generally, for any $m \times n$ matrix $A$, the set $\ker(A) = \{x \in \mathbb{R}^n : Ax = 0\}$ is a subspace of $\mathbb{R}^n$. This is the motivating example from Section 1.

Solution space of a homogeneous linear ODE. The set of solutions to $y'' + y = 0$ is a subspace of $C(\mathbb{R})$: if $f$ and $g$ solve the equation, then $\lambda f + \mu g$ does too, because differentiation is linear. (IOU: we will study the full structure of ODE solution spaces in Phase 8.)

Historical note

Hermann Grassmann published his Ausdehnungslehre (Theory of Extension) in 1844, giving the first abstract treatment of vector spaces in arbitrary dimensions. The mathematical community almost completely ignored it — the notation was unfamiliar and the ideas too far ahead of their time. Peano (1888) gave the modern axiomatic definition in his Calcolo geometrico. The concept crystallised slowly through the early 20th century in the work of Steinitz (1913), Banach (1920s), and the Bourbaki group. Today it is the language of all mathematics.

What breaks

Drop closure under addition. The unit sphere $S = \{v \in \mathbb{R}^3 : \|v\| = 1\}$ is closed under scalar multiplication by $\pm 1$, but $(1,0,0) + (0,1,0) = (1,1,0)$ has norm $\sqrt{2} \neq 1$. Not a subspace.

Missing zero vector. The plane $x + y + z = 1$ in $\mathbb{R}^3$ does not contain the origin (since $0 + 0 + 0 = 0 \neq 1$). It "looks flat" but is an affine subspace, not a vector subspace. If you take two points on it and add them, the result satisfies $x + y + z = 2$, not $1$. Closure fails.

Drop the field. If you try to build a "vector space" over $\mathbb{Z}$ (the integers), you get a module, not a vector space. Modules lack division by scalars, and many key theorems (every subspace has a complement, every linearly independent set extends to a basis) fail. The theory is vastly more complicated.

A vector space is any set where you can add elements and scale them, with the results staying in the set — eight axioms capture this, and everything else follows.
